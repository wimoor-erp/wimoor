# 查凭证模块功能解析文档

## 1. 模块概述

查凭证模块是财务管理系统中用于查询、浏览和管理记账凭证的核心功能模块。该模块采用前后端分离架构，前端使用Vue 3 + Element Plus实现，后端基于Spring Boot + MyBatis-Plus构建，提供了全面的凭证查询、管理和操作功能。

### 1.1 核心功能

- **多条件查询**：支持按凭证期间、凭证字号、记录摘要、会计科目、制单人等多维度筛选
- **智能列表展示**：以表格形式展示凭证及其分录信息，支持跨行合并显示
- **批量操作**：支持批量审核凭证
- **数据导出**：支持导出凭证列表为Excel文件
- **凭证管理**：集成编辑、删除等管理功能
- **状态可视化**：清晰展示凭证的不同状态（草稿、待审核、已审核、已过账、已作废）

### 1.2 技术架构

- **前端**：Vue 3、Composition API、Element Plus、Axios
- **后端**：Spring Boot 2.x、MyBatis-Plus、Spring Security
- **数据库**：MySQL
- **API风格**：RESTful API
- **数据传输**：JSON格式

### 2.3 核心逻辑

#### 数据加载逻辑

```javascript
async function getList() {
  loading.value = true
  queryParams.value.groupid=await finStore.getCurrentTenantId();
  listVouchers(queryParams.value).then(response => {
    vouchersList.value = [];
    response.rows.forEach(row=>{
      var myrow=row.entries;
      var length=row.entries.length;
      myrow.forEach((item,index)=>{
        const targetObj =item;
        const sourceObj =row;
        const excludeProp = 'entries';
        
        // 创建源对象的副本
        const sourceCopy = { ...sourceObj };
        
        // 删除要避开的属性
        delete sourceCopy[excludeProp];
        
        // 使用Object.assign合并
        const result = Object.assign({}, targetObj, sourceCopy);
        if(index==0){
          result.rowspan=length;
        }
        vouchersList.value.push(result);
      })
    });
    total.value = response.total
    loading.value = false
  })
}
```

**关键实现**：
- 获取当前租户ID并添加到查询参数
- 调用`listVouchers`API获取凭证列表
- 处理响应数据，将凭证主表数据与分录数据合并
- 为每条凭证的第一条分录设置跨行合并参数
- 更新表格数据和总数

```java
@Override
public int approveFinVouchers(List<Long> finVoucherIds) {
    //批量审核通过
    UserInfo userinfo = UserInfoContext.get();
    for(Long finVoucherId:finVoucherIds){
        FinVouchers voucher = this.selectFinVouchersByVoucherId(finVoucherId);
        if(voucher!=null && voucher.getVoucherStatus()==1){
            voucher.setVoucherStatus(2); // 已审核
            voucher.setUpdatedTime(new Date());
            voucher.setUpdateBy(userinfo.getUserName());
            finVouchersMapper.updateFinVouchers(voucher);
            // 不需要再次更新账簿！
        }
    }
    return 1;
}
```

**关键实现**：

- 遍历凭证ID列表，逐个处理
- 查询凭证详情
- 检查凭证状态是否为待审核

   - 接收响应数据
   - 处理凭证主表与分录数据
   - 实现跨行合并显示
   - 更新表格数据

### 5.2 批量审核流程

1. **前端发起请求**：用户选择凭证，点击审核按钮
2. **确认操作**：前端弹出确认对话框

4. **后端处理**：
   - 控制器接收请求，检查参数
   - 调用服务层批量审核凭证
   - 更新凭证状态
5. **前端处理**：
   - 接收响应数据
   - 显示审核成功提示
   - 刷新凭证列表

### 5.3 凭证导出流程

1. **前端发起请求**：用户点击导出按钮
2. **参数处理**：前端封装查询条件
