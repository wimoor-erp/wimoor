# 1688绑定模块帮助文档

## 1. 模块概述

1688绑定模块是Wimoor系统中用于绑定和管理阿里巴巴1688平台账号的功能模块，通过该模块用户可以授权绑定1688账号，实现与1688平台的对接，方便后续的采购操作和物流信息查询。

## 2. 功能说明

### 2.1 核心功能

- **账号绑定**：支持绑定多个1688账号，每个账号可以设置自定义名称
- **授权管理**：处理1688平台的OAuth2授权流程，获取访问令牌
- **令牌管理**：自动管理访问令牌的刷新和有效期
- **账号列表**：展示已绑定的1688账号列表，包含账号状态和到期时间
- **账号编辑**：支持修改已绑定账号的名称和开发者信息
- **账号解绑**：支持删除已绑定的1688账号
- **延期授权**：支持对即将过期的账号进行授权延期

### 2.2 高级功能

- **开发者模式**：支持使用自定义的AppKey和AppSecret进行绑定
- **物流信息查询**：授权后可在采购单中查看1688平台的物流信息
- **订单管理**：可通过绑定的账号进行1688订单的相关操作

## 3. 使用流程

### 3.1 首次绑定1688账号

1. **进入模块**：在系统菜单中找到"采购" -> "1688绑定"模块
2. **点击绑定**：点击页面左上角的"绑定账号"按钮
3. **填写信息**：
   - 输入采购账号名称（必填）
   - 选择授权类型（常规或高级）
   - 若选择高级模式，需输入自定义的AppKey和AppSecret
4. **点击授权**：点击"授权"按钮
5. **1688授权**：系统会跳转到1688平台的授权页面，登录1688账号并确认授权
6. **授权完成**：授权成功后系统会自动跳转回绑定页面，并显示绑定成功提示

### 3.2 查看已绑定账号

1. **进入模块**：在系统菜单中找到"采购" -> "1688绑定"模块
2. **查看列表**：页面会显示已绑定的1688账号列表，包含以下信息：
   - 采购账号名称
   - 采购平台（1688）
   - 平台账户名称
   - 到期时间
   - 状态（已绑定/未绑定）
   - 操作按钮

### 3.3 编辑已绑定账号

1. **进入模块**：在系统菜单中找到"采购" -> "1688绑定"模块
2. **点击编辑**：在账号列表中，点击对应账号名称右侧的编辑图标
3. **修改信息**：修改采购账号名称或开发者信息
4. **点击保存**：点击"保存"按钮完成修改

### 3.4 延期授权

1. **进入模块**：在系统菜单中找到"采购" -> "1688绑定"模块
2. **点击延期**：在账号列表中，点击对应账号的"延期授权"按钮
3. **1688授权**：系统会跳转到1688平台的授权页面，登录1688账号并确认授权
4. **授权完成**：授权成功后系统会自动跳转回绑定页面，并显示授权成功提示

### 3.5 删除已绑定账号

1. **进入模块**：在系统菜单中找到"采购" -> "1688绑定"模块
2. **点击删除**：在账号列表中，点击对应账号的"删除"按钮
3. **确认删除**：系统会提示确认删除，点击确认后完成解绑

## 4. 常见问题

### 4.1 授权失败怎么办？

- 检查网络连接是否正常
- 确认1688账号登录状态是否有效
- 检查AppKey和AppSecret是否正确（高级模式）
- 尝试重新绑定操作

### 4.2 令牌过期怎么办？

- 系统会自动管理令牌的刷新
- 若长效令牌（refresh_token）过期，需要重新进行绑定操作
- 长效令牌的有效期为半年

### 4.3 如何查看物流信息？

- 授权绑定1688账号后，在采购单详情页面可以查看对应的物流信息
- 系统会自动从1688平台获取物流跟踪数据

### 4.4 可以绑定多个1688账号吗？

- 是的，系统支持绑定多个1688账号
- 每个账号需要单独进行授权操作

## 5. 注意事项

### 5.1 安全注意事项

- 请勿使用他人的1688账号进行绑定
- 保护好您的AppKey和AppSecret信息，避免泄露
- 定期检查已绑定账号的状态和有效期

### 5.2 操作注意事项

- 绑定账号时请确保网络连接稳定
- 授权过程中请勿关闭浏览器窗口
- 若遇到授权页面无法打开，请检查网络设置或尝试更换浏览器

### 5.3 技术注意事项

- 高级模式需要您在1688开放平台注册开发者账号并创建应用
- 自定义的AppKey和AppSecret需要具有相应的API权限
- 系统通过聚石塔服务与1688平台进行通信，请确保网络可以访问相关服务
