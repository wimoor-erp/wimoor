# 查凭证模块帮助文档

## 1. 模块概述

查凭证模块是财务管理系统中用于查询、浏览和管理记账凭证的功能模块。该模块提供了全面的凭证查询功能，支持多条件筛选、导出、审核等操作，方便用户快速找到所需的凭证信息并进行相关处理。

### 1.1 核心功能

- **凭证查询**：支持按凭证期间、凭证字号、记录摘要、会计科目、制单人等多条件查询
- **凭证列表**：以表格形式展示凭证及其分录信息，支持跨行合并显示
- **批量操作**：支持批量审核凭证
- **凭证导出**：支持导出凭证列表为Excel文件
- **凭证管理**：支持查看、编辑、删除凭证
- **状态显示**：清晰展示凭证的不同状态（草稿、待审核、已审核、已过账、已作废）

### 1.2 技术特点

- **前后端分离**：前端使用Vue 3 + Element Plus实现，后端基于Spring Boot构建
- **响应式设计**：适配不同屏幕尺寸
- **数据联动**：凭证主表与分录数据联动显示
- **智能筛选**：支持多条件组合筛选
- **高效渲染**：优化大数据量下的表格渲染性能

## 2. 功能说明

### 2.1 凭证查询

**功能描述**：根据用户设置的条件查询记账凭证列表。

**查询条件**：
- **凭证期间**：选择凭证日期范围
- **凭证字号**：输入凭证编号
- **记录摘要**：输入摘要关键字
- **会计科目**：选择会计科目
- **制单人**：输入制单人姓名

**操作步骤**：
1. 进入查凭证页面
2. 设置查询条件
3. 点击"搜索"按钮执行查询
4. 查看查询结果

**高级搜索**：
点击搜索框右侧的筛选按钮，展开高级搜索面板，可以设置更多查询条件。

### 2.2 凭证列表

**功能描述**：以表格形式展示凭证及其分录信息。

**列信息**：
- **复选框**：用于选择凭证进行批量操作
- **操作**：编辑、删除按钮
- **凭证日期**：凭证的制单日期
- **凭证字号**：凭证的编号，如"记-001"
- **摘要**：凭证分录的摘要说明
- **科目**：会计科目名称
- **借方金额**：借方发生额
- **贷方金额**：贷方发生额
- **附件数量**：凭证附件的数量
- **原单据编号**：原始单据的编号
- **制单人**：凭证的制单人
- **审核人**：凭证的审核人
- **状态**：凭证的当前状态
- **备注**：凭证的备注信息
- **制单日期**：凭证的创建日期

**特殊显示**：
- 同一凭证的多条分录会进行跨行合并显示，减少重复信息
- 凭证状态以文字形式清晰展示

### 2.3 批量审核

**功能描述**：批量审核选中的凭证。

**操作步骤**：
1. 在凭证列表中勾选需要审核的凭证
2. 点击"审核"按钮
3. 在弹出的确认对话框中点击"确定"
4. 系统自动审核选中的凭证
5. 审核完成后，凭证状态会更新为"已审核"

**注意事项**：
- 只有状态为"草稿"或"待审核"的凭证才能进行审核
- 审核后的凭证才能进行过账操作

### 2.4 凭证导出

**功能描述**：导出凭证列表为Excel文件。

**操作步骤**：
1. 设置查询条件（可选）
2. 点击"导出"按钮
3. 系统自动生成Excel文件并下载

**导出内容**：
- 凭证基本信息
- 凭证分录信息
- 凭证状态信息

### 2.5 凭证管理

**功能描述**：对凭证进行查看、编辑、删除等操作。

**编辑凭证**：
1. 在凭证列表中找到需要编辑的凭证
2. 点击"编辑"按钮
3. 系统跳转到凭证编辑页面
4. 修改凭证信息
5. 保存修改

**删除凭证**：
1. 在凭证列表中找到需要删除的凭证
2. 点击"删除"按钮
3. 在弹出的确认对话框中点击"确定"
4. 系统删除该凭证

**注意事项**：
- 已过账的凭证需要先反过账才能修改或删除
- 已结账期间的凭证不能修改或删除

## 3. 操作指南

### 3.1 基本查询操作

**步骤1：进入查凭证页面**
- 从系统菜单中选择"财务管理" > "凭证管理" > "查凭证"

**步骤2：设置查询条件**
- **凭证期间**：点击日期选择器，选择开始和结束月份
- **高级搜索**：点击筛选按钮，展开高级搜索面板，设置更多查询条件

**步骤3：执行查询**
- 点击"搜索"按钮，系统根据设置的条件查询凭证
- 点击"重置"按钮，清空所有查询条件

**步骤4：查看凭证列表**
- 系统以表格形式展示查询结果
- 可以通过分页控件切换页码和调整每页显示数量

### 3.2 批量审核操作

**步骤1：选择凭证**
- 在凭证列表的复选框中勾选需要审核的凭证
- 可以通过表头的复选框全选当前页的凭证

**步骤2：执行审核**
- 点击页面顶部的"审核"按钮
- 在弹出的确认对话框中点击"确定"

**步骤3：查看审核结果**
- 审核完成后，系统会提示审核成功
- 凭证状态会更新为"已审核"

### 3.3 凭证导出操作

**步骤1：设置查询条件**
- 根据需要设置凭证的查询条件

**步骤2：执行导出**
- 点击页面顶部的"导出"按钮
- 系统自动生成Excel文件并开始下载

**步骤3：打开导出文件**
- 下载完成后，打开Excel文件查看导出的凭证数据

### 3.4 凭证编辑操作

**步骤1：找到需要编辑的凭证**
- 在凭证列表中找到需要编辑的凭证

**步骤2：进入编辑页面**
- 点击凭证行的"编辑"按钮
- 或点击凭证字号链接

**步骤3：修改凭证信息**
- 在编辑页面修改凭证的基本信息和分录
- 确保借贷金额平衡

**步骤4：保存修改**
- 点击"保存"按钮提交修改
- 系统会自动更新凭证数据

### 3.5 凭证删除操作

**步骤1：找到需要删除的凭证**
- 在凭证列表中找到需要删除的凭证

**步骤2：执行删除**
- 点击凭证行的"删除"按钮
- 在弹出的确认对话框中点击"确定"

**步骤3：查看删除结果**
- 删除完成后，系统会提示删除成功
- 凭证会从列表中消失

## 4. 常见问题

### 4.1 无法找到需要的凭证

**问题描述**：设置了查询条件但找不到需要的凭证。

**解决方案**：
- 检查查询条件是否设置正确
- 确认凭证的日期范围是否在查询条件内
- 尝试使用更宽泛的查询条件
- 检查凭证是否已被删除或作废

### 4.2 批量审核失败

**问题描述**：批量审核凭证时提示失败。

**解决方案**：
- 检查选中的凭证状态是否为"草稿"或"待审核"
- 确认网络连接是否正常
- 尝试单个审核凭证，查看具体失败原因
- 联系系统管理员检查系统日志

### 4.3 导出文件为空

**问题描述**：点击导出按钮后下载的Excel文件为空。

**解决方案**：
- 检查查询条件是否过于严格，导致没有符合条件的凭证
- 确认当前查询结果是否有数据
- 尝试减少查询条件，扩大查询范围
- 检查系统权限是否允许导出操作

### 4.4 无法编辑凭证

**问题描述**：点击编辑按钮后无法修改凭证。

**解决方案**：
- 检查凭证状态，已过账的凭证需要先反过账
- 确认凭证所在会计期间是否已结账
- 检查用户是否有编辑凭证的权限
- 联系系统管理员检查系统设置

### 4.5 凭证列表显示异常

**问题描述**：凭证列表显示不完整或格式异常。

**解决方案**：
- 刷新页面重新加载数据
- 清除浏览器缓存后重新登录
- 检查网络连接是否稳定
- 尝试调整浏览器窗口大小

## 5. 最佳实践

### 5.1 查询技巧

1. **精确查询**：使用凭证字号进行精确查询
2. **范围查询**：使用凭证期间选择器进行日期范围查询
3. **组合查询**：结合多个条件进行精准筛选
4. **常用条件**：保存常用的查询条件组合
5. **模糊查询**：在摘要中使用关键字进行模糊查询

### 5.2 操作建议

1. **定期审核**：定期批量审核凭证，避免凭证积压
2. **及时导出**：定期导出凭证数据作为备份
3. **分类管理**：根据业务类型和时间对凭证进行分类管理
4. **权限控制**：合理设置用户权限，确保凭证数据安全
5. **操作记录**：保持良好的操作习惯，记录重要的凭证操作

### 5.3 性能优化

1. **合理设置查询条件**：避免使用过于宽泛的查询条件
2. **分页查询**：使用分页功能，避免一次性加载过多数据
3. **定期清理**：定期清理无效的凭证数据
4. **浏览器优化**：使用现代浏览器，保持浏览器更新
5. **网络环境**：确保网络连接稳定，避免在网络不稳定时进行大量操作

### 5.4 数据安全

1. **权限管理**：严格控制凭证查询和修改权限
2. **操作日志**：系统自动记录凭证操作日志
3. **数据备份**：定期备份凭证数据
4. **防篡改**：防止未经授权修改凭证数据
5. **敏感信息**：对凭证中的敏感信息进行适当处理
