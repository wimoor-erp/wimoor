# 财务-账期末结算模块帮助文档

## 1. 模块概述

财务-账期末结算模块（回款记录）是一个专为财务人员设计的工具，用于从亚马逊直接拉取结算报表并展示每笔订单的结算时间和费用等信息。通过本模块，财务人员可以：

- 查看亚马逊平台的结算记录
- 按结算状态、店铺、国家、日期等多维度筛选数据
- 切换不同币种查看结算金额
- 下载单个账期的明细和导出所有账期数据
- 查看账期的详细费用构成
- 进行账期汇总校验和重算

## 2. 功能说明

### 2.1 结算状态筛选

**功能描述**：筛选已结算和未结算的记录。

**操作步骤**：
1. 进入回款记录页面
2. 在页面顶部的单选按钮组中选择"已结算"或"未结算"
3. 系统会自动更新结算记录列表

### 2.2 多维度查询

**功能描述**：按店铺、国家、结算日期、转账日期等多维度筛选结算记录。

**操作步骤**：
1. 选择店铺分组和国家
2. 选择日期类型（结算日期/转账日期）
3. 选择日期范围
4. 点击查询按钮查看结算记录

### 2.3 币种切换

**功能描述**：在原币种、CNY、USD之间切换查看结算金额。

**操作步骤**：
1. 在页面顶部的币种选择下拉框中选择所需币种
2. 系统会自动更新所有金额的显示

### 2.4 数据汇总

**功能描述**：查看未结算金额、已结算金额、总金额等汇总数据。

**操作步骤**：
1. 进入回款记录页面
2. 页面顶部的卡片会显示当前筛选条件下的汇总数据

### 2.5 账期汇总

**功能描述**：支持校验账期汇总和强制重算功能。

**操作步骤**：
1. 选择"已结算"状态
2. 点击"汇总"按钮，选择"校验账期汇总"或"强制重算"
3. 系统会执行相应的汇总操作

### 2.6 明细下载

**功能描述**：支持下载单个账期的明细和导出所有账期数据。

**操作步骤**：
1. 对于单个账期，点击操作列的"下载明细"按钮
2. 对于所有账期，点击页面顶部的"导出所有账期"按钮
3. 系统会生成并下载Excel格式的结算明细报表

### 2.7 账期详情查看

**功能描述**：查看账期的详细费用构成。

**操作步骤**：
1. 点击操作列的"查看详情"按钮
2. 系统会弹出账期详情对话框，显示销售汇总和详细费用构成

## 3. 操作指南

### 3.1 基本操作流程

1. **进入模块**：从左侧菜单导航到"亚马逊 > 回款记录"
2. **设置查询条件**：
   - 选择结算状态（已结算/未结算）
   - 选择店铺分组和国家
   - 选择币种
   - 选择日期类型和日期范围
3. **查看结算记录**：系统会显示符合条件的结算记录列表
4. **查看汇总数据**：页面顶部的卡片会显示汇总数据
5. **执行汇总操作**：点击"汇总"按钮，选择相应的操作
6. **下载明细**：点击"下载明细"按钮，下载单个账期的明细
7. **导出所有账期**：点击"导出所有账期"按钮，下载所有账期的数据
8. **查看详情**：点击"查看详情"按钮，查看账期的详细费用构成

### 3.2 常见问题及解决方法

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 无法选择"原币种" | 未选择国家 | 先选择店铺分组和国家，再选择"原币种" |
| 导出所有账期速度慢 | 选择的时间范围过大 | 建议不要导出超过三个账期的数据 |
| 账期汇总失败 | 网络连接问题或数据异常 | 检查网络连接，或尝试强制重算 |
| 结算记录显示不全 | 筛选条件设置不当 | 调整筛选条件，确保日期范围正确 |

## 4. 注意事项

1. **数据准确性**：系统从亚马逊直接拉取结算数据，但由于网络延迟或亚马逊API限制，数据可能存在短暂的不一致，建议定期进行账期汇总校验。

2. **导出限制**：导出所有账期数据时，由于数据量较大，可能需要较长时间，请耐心等待。

3. **币种转换**：币种转换基于系统内置的汇率数据，可能与实际汇率存在轻微差异。

4. **权限控制**：只有具有财务权限的用户才能访问本模块，请确保您的账号具有相应权限。

5. **数据安全**：本模块包含敏感的财务数据，请妥善保管您的账号和密码，避免数据泄露。

## 5. 术语解释

- **结算周期**：亚马逊平台的结算时间段，通常为两周。
- **转账日期**：亚马逊向卖家账户转账的日期。
- **已结算**：亚马逊已经完成转账的账期。
- **未结算**：亚马逊尚未完成转账的账期。
- **账期汇总**：对一个账期内的所有交易进行汇总计算，确保数据准确性。
- **原币种**：亚马逊平台所在国家的货币。
