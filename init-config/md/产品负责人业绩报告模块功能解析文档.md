# 产品负责人业绩报告模块功能解析文档

## 1. 模块概述

**产品负责人业绩报告模块**是一个专为产品负责人设计的业绩统计工具，用于核算业绩和提成。该模块通过整合亚马逊平台的销售数据、成本数据和费用数据，为每个产品负责人提供详细的业绩分析，帮助管理层准确计算绩效和提成。

### 1.1 核心功能
- **业绩数据统计**：按产品负责人统计销售额、销量、订单量等核心指标
- **成本费用分析**：计算采购成本、运费、广告费等各项费用
- **利润核算**：自动计算毛利润和毛利率
- **多维度筛选**：支持按日期、店铺、站点、产品负责人等维度筛选
- **多币种转换**：支持人民币、美元和站点币种的切换
- **数据导出**：支持导出Excel格式的业绩报表

### 1.2 技术架构
- **前端**：Vue 3 + Element Plus + Axios
- **后端**：Spring Boot 2.x + MyBatis-Plus
- **数据库**：MySQL
- **核心算法**：Aviator表达式引擎（用于利润计算）

## 5. 业务逻辑流程

### 5.1 业绩数据查询流程
1. 前端发送查询请求，包含筛选条件
2. 后端接收请求，解析参数
3. 构建查询条件，调用服务层方法
4. 服务层查询数据库，获取业绩数据
5. 计算退款率、利润率等衍生指标
6. 汇总数据，返回结果
7. 前端接收数据，渲染表格

### 5.2 业绩数据导出流程
1. 前端发送导出请求，包含筛选条件
2. 后端接收请求，解析参数
3. 构建查询条件，调用服务层方法
4. 服务层查询数据库，获取业绩数据
5. 生成Excel文件，写入数据
6. 设置响应头，返回Excel文件
7. 前端接收文件，触发下载

### 5.3 业绩数据刷新流程
1. 前端发送刷新请求，包含日期范围
2. 后端接收请求，解析参数
3. 创建异步线程，执行数据刷新
4. 调用Amazon Settlement Analysis服务，重新计算业绩数据
5. 清除缓存，确保数据更新
6. 返回进度标识，用于前端跟踪刷新状态

### 5.4 币种转换流程
1. 前端选择目标币种
2. 后端根据源币种和目标币种，调用汇率转换服务
3. 对各项金额指标进行汇率转换
4. 重新计算利润和利润率
5. 返回转换后的数据
