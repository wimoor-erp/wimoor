# 账期末结算模块帮助文档

## 1. 模块概述

账期末结算模块是一个用于统计和管理亚马逊平台未结算订单费用的功能模块，主要用于查看每笔订单产生的时间、数量、费用类型、币种等详细信息，并支持导出报表功能。

## 2. 功能说明

### 2.1 核心功能

- **数据展示**：展示未结算的订单费用明细，包括商品信息、费用时间、订单号、数量、费用金额、币种、费用类型、事件类型等
- **筛选功能**：支持按日期范围和 marketplace 进行数据筛选
- **导出功能**：支持导出未结算费用报表为 Excel 格式

### 2.2 界面元素

| 元素名称 | 功能描述 | 操作说明 |
|---------|---------|--------|
| 店铺分组选择器 | 选择要查看的店铺分组 | 点击下拉菜单选择分组 |
| 日期选择器 | 选择日期范围 | 点击弹出日期选择面板，选择开始和结束日期 |
| 导出按钮 | 导出未结算费用报表 | 点击按钮后会下载 Excel 报表 |
| 数据表格 | 展示未结算费用明细 | 可点击表头进行排序，支持分页浏览 |

### 2.3 表格字段说明

| 字段名称 | 说明 | 数据来源 |
|---------|-----|----------|
| 平台商品信息 | 显示商品图片、名称、SKU、ASIN、站点 | 从产品信息和市场信息中获取 |
| 费用时间 | 显示费用产生的时间和订单号 | 从结算数据中获取 |
| 数量 | 显示订单数量 | 从结算数据中获取 |
| 费用 | 显示费用金额 | 从结算数据中获取 |
| 费用币种 | 显示费用的货币类型 | 从结算数据中获取 |
| 费用类型 | 显示费用类型及其中文名称 | 从结算数据和费用类型描述中获取 |
| 事件类型 | 显示事件类型 | 从结算数据中获取 |

## 3. 使用指南

### 3.1 基本操作流程

1. **进入模块**：从左侧菜单导航至 "亚马逊 > 报表 > 账期末结算" 模块
2. **选择筛选条件**：
   - 选择店铺分组
   - 选择日期范围
3. **查看数据**：系统会自动加载符合条件的未结算费用数据
4. **导出报表**：点击 "导出" 按钮，系统会生成并下载 Excel 格式的报表

### 3.2 常见操作场景

#### 场景一：查看特定时间段的未结算费用

1. 在日期选择器中选择开始日期和结束日期
2. 系统会自动更新表格数据，显示该时间段内的未结算费用

#### 场景二：查看特定店铺的未结算费用

1. 在店铺分组选择器中选择特定的店铺分组
2. 系统会自动更新表格数据，显示该店铺分组的未结算费用

#### 场景三：导出未结算费用报表

1. 设置好筛选条件
2. 点击 "导出" 按钮
3. 系统会生成 Excel 报表并自动下载
4. 打开下载的 Excel 文件查看详细数据

## 4. 常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 数据加载缓慢 | 数据量较大 | 尝试缩小日期范围或选择特定店铺分组 |
| 导出失败 | 网络问题或数据量过大 | 检查网络连接，或尝试分批次导出 |
| 数据显示不完整 | 筛选条件设置不当 | 检查日期范围和店铺分组设置 |
| 费用类型显示为英文 | 系统配置问题 | 联系管理员检查费用类型描述配置 |

## 5. 注意事项

- 账期末结算模块仅显示未结算的费用数据，已结算的数据不会显示在此模块
- 导出报表时，系统会根据当前设置的筛选条件生成报表
- 数据加载速度取决于数据量大小，建议合理设置筛选条件
- 如有数据异常，请联系系统管理员进行排查
