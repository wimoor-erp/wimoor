# 发货详情报表模块帮助文档

## 1. 模块概述

发货详情报表模块是亚马逊FBA发货管理系统的重要组成部分，用于查看和分析所有FBA货件的详细信息。该模块提供两种数据视图：按货件汇总和按SKU汇总，帮助用户全面了解货件的发货、接收、物流费用等关键信息。

### 1.1 主要功能

- **货件汇总视图**：以货件为单位展示发货信息，包括货件状态、日期、收货站点、承运商、接收数量、物流费用、运输重量等
- **SKU汇总视图**：以SKU为单位展示发货明细，包括SKU、货件编码、发货店铺、收货站点、发货仓库、发货日期、发货数量、到货数量等
- **多维度筛选**：支持按日期类型、仓库、货件编码、接收异常、承运商等条件筛选数据
- **数据导出**：支持多种格式的Excel导出，包括普通导出、含子SKU、发货数量简约版、发货处理任务量
- **数据统计**：自动汇总总发货数量和总接收数量

### 1.2 适用场景

- 查看特定时间段内的所有FBA货件发货情况
- 分析货件的接收异常情况
- 统计物流费用和运输重量
- 按SKU查看发货明细和到货情况
- 导出数据进行进一步分析

## 2. 界面说明

### 2.1 页面布局

页面采用标准的报表布局，包含以下主要区域：

1. **顶部标签页**：切换"按货件汇总"和"按SKU汇总"两种视图
2. **筛选条件区**：包含店铺选择、日期类型选择、日期范围选择、仓库选择、货件编码搜索、高级筛选
3. **操作按钮区**：包含数据统计显示、导出按钮、自定义列按钮、帮助文档按钮
4. **数据表格区**：展示货件或SKU的详细信息

## 3. 操作指南

### 3.1 切换视图模式

1. 在页面顶部的标签页中，点击"按货件汇总"或"按SKU汇总"
2. 系统会自动切换到对应的视图模式并重新加载数据

### 3.2 设置筛选条件

#### 3.2.1 选择店铺

1. 点击"店铺"下拉选择框
2. 选择要查看的店铺
3. 系统会自动加载该店铺的货件数据

#### 3.2.2 选择日期类型

1. 点击"日期类型"下拉选择框
2. 选择"创建日期"或"发货日期"
   - 创建日期：按货件创建时间筛选
   - 发货日期：按货件实际发货时间筛选

#### 3.2.3 选择日期范围

1. 点击日期选择器
2. 选择开始日期和结束日期
3. 系统会自动筛选指定日期范围内的数据

#### 3.2.4 选择仓库

1. 点击"仓库"下拉选择框
2. 选择要查看的仓库
3. 默认为"全部"，显示所有仓库的数据

#### 3.2.5 搜索货件编码

1. 在"货件编码"输入框中输入货件编码
2. 点击搜索按钮或按回车键
3. 系统会显示匹配的货件数据

#### 3.2.6 高级筛选

1. 点击"筛选"按钮打开高级筛选面板
2. 设置以下筛选条件：
   - **接收异常**：选择"全部"、"是"或"否"
     - 是：只显示接收数量与发货数量不一致的货件
     - 否：只显示接收数量与发货数量一致的货件
   - **选择承运商**：选择特定的承运商
3. 点击"搜索"按钮应用筛选条件
4. 点击"取消"按钮关闭筛选面板

### 3.3 查看数据统计

在按SKU汇总视图下，页面右上角会显示以下统计信息：

- **总发货**：所有货件的总发货数量
- **总接收**：所有货件的总接收数量

### 3.4 导出数据

#### 3.4.1 按货件汇总视图导出

1. 点击"导出"按钮，默认导出货件汇总数据
2. 点击导出按钮右侧的下拉箭头，可选择以下导出格式：
   - **发货数量简约版**：只包含发货数量相关信息的简化版本
   - **发货处理任务量**：包含发货处理任务量的统计数据

#### 3.4.2 按SKU汇总视图导出

1. 点击"导出"按钮，默认导出SKU汇总数据
2. 点击导出按钮右侧的下拉箭头，可选择以下导出格式：
   - **含子SKU**：包含子SKU信息的详细版本
   - **发货数量简约版**：只包含发货数量相关信息的简化版本
   - **发货处理任务量**：包含发货处理任务量的统计数据

#### 3.4.3 导出文件说明

导出的Excel文件包含以下信息：

**按货件汇总导出**：
- 货件编码、发货店铺、收货站点、配送中心、发货仓库
- 创建日期、发货日期、预计到货日期、开始接受日期、完成日期
- 承运商、运输方式、发货渠道
- 发货数量、到货数量
- 运输费用、其它费用、物流费用统计
- 实际计费重量、预估发货重量、装箱实际重量、装箱材积重量
- 重量单位、货件状态

**按SKU汇总导出**：
- SKU、货件编码、内部编号
- 发货店铺、收货站点、配送中心、发货仓库
- 发货日期、预计到货日期
- 发货渠道、渠道名称
- 发货数量、到货数量
- 货件状态、创建日期

### 3.5 自定义列显示

1. 点击"自定义列"按钮
2. 在弹出的对话框中勾选或取消勾选要显示的列
3. 点击"确定"保存设置
4. 表格会按照设置显示相应的列

### 3.6 排序数据

1. 点击表格列标题
2. 系统会按照该列进行升序或降序排序
3. 再次点击可切换排序方向

### 3.7 分页浏览

1. 在表格底部显示当前页码和总页数
2. 点击页码或使用上一页/下一页按钮浏览数据
3. 可以设置每页显示的数据条数

## 4. 常见问题

### 4.1 为什么看不到某些货件？

可能的原因：
- 货件不在选择的日期范围内
- 货件不在选择的仓库中
- 货件不在选择的店铺中
- 货件编码搜索条件不匹配
- 高级筛选条件排除了该货件

解决方法：
- 检查并调整筛选条件
- 清空货件编码搜索框
- 将仓库设置为"全部"
- 将店铺设置为所有店铺

### 4.2 接收异常是什么意思？

接收异常是指货件的到货数量与发货数量不一致的情况。这可能由以下原因造成：
- 运输过程中的丢失或损坏
- 亚马逊接收时的数量差异
- 数据同步延迟

### 4.3 如何查看有接收异常的货件？

1. 点击"筛选"按钮打开高级筛选面板
2. 在"接收异常"下拉框中选择"是"
3. 点击"搜索"按钮
4. 系统会显示所有有接收异常的货件

### 4.4 导出的数据为什么是空的？

可能的原因：
- 当前筛选条件下没有数据
- 日期范围设置不正确
- 仓库或店铺选择不正确

解决方法：
- 检查并调整筛选条件
- 扩大日期范围
- 将仓库设置为"全部"

### 4.5 数据统计不准确怎么办？

可能的原因：
- 数据同步延迟
- 筛选条件设置不正确
- 系统缓存问题

解决方法：
- 刷新页面重新加载数据
- 检查筛选条件
- 联系技术支持

### 4.6 如何快速找到特定货件？

方法一：使用货件编码搜索
1. 在"货件编码"输入框中输入货件编码
2. 点击搜索按钮

方法二：使用日期筛选
1. 设置货件的创建日期或发货日期范围
2. 系统会显示该时间段内的所有货件
3. 在表格中查找目标货件

### 4.7 导出的Excel文件打不开？

可能的原因：
- 文件损坏
- Excel版本不支持

解决方法：
- 重新导出数据
- 使用较新版本的Excel打开
- 联系技术支持

### 4.8 数据更新不及时？

可能的原因：
- 系统数据同步延迟
- 浏览器缓存问题

解决方法：
- 刷新页面
- 清除浏览器缓存
- 等待数据同步完成
