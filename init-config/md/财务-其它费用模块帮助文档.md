# 财务-其它费用模块帮助文档

## 1. 模块概述

财务-其它费用模块是财务管理系统中的重要功能模块，用于管理和跟踪企业的其他费用支出，包括各种非经常性或特殊费用项目。该模块支持费用的添加、导入、查询、导出等功能，帮助企业全面掌握和分析费用支出情况。

### 1.1 主要功能

- 添加和管理其他费用项目
- 导入费用数据（支持Excel批量导入）
- 按店铺、站点、费用类型、日期等维度筛选费用数据
- 查看费用明细和汇总数据
- 导出费用数据为Excel文件（支持每天详细和按月汇总两种格式）
- 管理费用类型

### 1.2 适用场景

- 财务人员记录和管理企业的其他费用支出
- 管理层分析企业费用支出情况，优化成本结构
- 审计人员审查企业费用支出的合理性和合规性
- 企业需要对特殊费用进行单独跟踪和管理

## 2. 操作指南

### 2.1 进入模块

1. 登录系统后，在左侧菜单中找到「财务」→「本地账目」→「其它费用」
2. 点击进入其它费用页面

### 2.2 查看费用列表

- 页面默认显示所有费用数据，按日期排序
- 每条费用记录包含以下信息：
  - 店铺
  - 国家/地区
  - 费用类型
  - 货币
  - 金额
- 点击费用记录前的展开按钮，可以查看该费用的分摊产品详情

### 2.3 筛选费用数据

1. **按店铺和站点筛选**：通过顶部的店铺选择器选择特定的店铺和站点
2. **按费用类型筛选**：在「请选择其它费用项」下拉菜单中选择特定的费用类型
3. **按日期筛选**：点击日期选择器，选择特定的日期范围
4. **按SKU搜索**：在搜索框中输入SKU，按回车键或点击搜索按钮
5. 点击「重置」按钮，清空所有筛选条件

### 2.4 导出费用数据

1. 设置好筛选条件（可选）
2. 点击页面顶部的「导出」按钮，选择导出格式：
   - 每天详细：导出每日的详细费用数据
   - 按月汇总：导出按月汇总的费用数据
3. 系统会生成并下载Excel格式的费用数据文件

### 2.5 添加费用

1. 点击页面顶部的「添加费用」按钮
2. 在弹出的对话框中填写以下信息：
   - 店铺
   - 站点
   - 费用类型
   - 日期
   - 货币
   - 金额
   - 分摊产品（可选）
3. 点击「确定」按钮，保存新的费用记录
4. 系统会自动刷新费用列表，显示新添加的记录

### 2.6 管理费用类型

1. 点击页面顶部的「费用类型管理」按钮
2. 在弹出的对话框中可以添加、编辑或删除费用类型
3. 点击「确定」按钮，保存费用类型的变更

### 2.7 导入分摊费

1. 点击页面顶部的「导入分摊费」按钮
2. 在弹出的对话框中下载导入模板
3. 按照模板格式填写费用数据
4. 上传填写好的Excel文件
5. 系统会自动导入费用数据并刷新列表

## 3. 常见问题

### 3.1 如何快速找到特定店铺的费用数据？

- 在店铺选择器中选择特定的店铺
- 系统会自动筛选并显示该店铺的费用数据

### 3.2 如何批量导入费用数据？

- 点击「导入分摊费」按钮
- 下载导入模板
- 按照模板格式填写费用数据
- 上传Excel文件
- 系统会自动导入并处理数据

### 3.3 如何查看费用的分摊详情？

- 在费用列表中找到需要查看的费用记录
- 点击该记录前的展开按钮
- 系统会显示该费用的分摊产品详情，包括产品图片、名称、SKU和分摊金额

### 3.4 如何导出按月汇总的费用数据？

- 点击「导出」按钮，选择「按月汇总」选项
- 系统会生成并下载按月汇总的费用数据文件

## 4. 注意事项

- 确保费用数据的准确性和完整性
- 定期备份费用数据，防止数据丢失
- 合理设置费用类型，便于费用分析和管理
- 遵守企业的财务制度和会计准则进行操作
- 对于大额费用，建议添加详细的备注说明
- 定期审查费用数据，及时发现和纠正错误
