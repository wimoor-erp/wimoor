# 总账目模块帮助文档

## 1. 模块概述

总账目模块是财务管理系统中的核心功能模块，用于记录和管理企业的总账数据，提供全面的财务核算和分析能力。该模块遵循会计核算原则，支持企业进行日常财务处理和财务报表生成。

### 1.1 主要功能

- 总账数据的查询和展示
- 按会计期间、创建时间等维度筛选总账数据
- 新增、修改、删除总账记录
- 导出总账数据为Excel文件
- 分页展示总账数据

### 1.2 适用场景

- 财务人员进行日常账务处理和核对
- 财务主管查看企业整体财务状况
- 审计人员进行财务审计
- 企业管理层分析财务数据，做出决策

## 2. 操作指南

### 2.1 进入模块

1. 登录系统后，在左侧菜单中找到「财务」→「账簿管理」→「总账目」
2. 点击进入总账目页面

### 2.2 查看总账列表

- 页面默认显示所有总账数据，按会计期间排序
- 每条总账记录包含以下信息：
  - 会计期间
  - 会计科目
  - 期初余额
  - 期初余额方向（借/贷）
  - 本期借方合计
  - 本期贷方合计
  - 期末余额
  - 期末余额方向（借/贷）
  - 创建时间

### 2.3 筛选总账数据

1. **按会计期间筛选**：在「会计期间」输入框中输入会计期间（格式：YYYYMM），如 "202511"
2. **按创建时间筛选**：点击「创建时间」选择器，选择特定日期
3. **按更新时间筛选**：点击「更新时间」选择器，选择特定日期
4. 点击「搜索」按钮，系统会根据筛选条件显示符合条件的总账数据
5. 点击「重置」按钮，清空所有筛选条件

### 2.4 导出总账数据

1. 设置好筛选条件（可选）
2. 点击页面右上角的「导出」按钮
3. 系统会生成并下载Excel格式的总账数据文件

### 2.5 新增总账记录

1. 点击页面顶部的「新增」按钮
2. 在弹出的对话框中填写以下信息：
   - 租户ID
   - 会计期间
   - 期初余额
   - 期初余额方向
   - 本期借方合计
   - 本期贷方合计
   - 期末余额
   - 期末余额方向
   - 创建时间
   - 更新时间
3. 点击「确定」按钮，保存新的总账记录
4. 系统会自动刷新总账列表，显示新添加的记录

### 2.6 修改总账记录

1. 在总账列表中找到需要修改的记录
2. 点击该记录操作列的「修改」按钮
3. 在弹出的对话框中修改相关信息
4. 点击「确定」按钮，保存修改
5. 系统会自动刷新总账列表，显示修改后的记录

### 2.7 删除总账记录

1. 在总账列表中找到需要删除的记录
2. 点击该记录操作列的「删除」按钮
3. 在弹出的确认对话框中点击「确定」
4. 系统会删除该总账记录并自动刷新列表

## 3. 常见问题

### 3.1 如何快速找到特定会计期间的总账数据？

- 在「会计期间」输入框中输入具体的会计期间，如 "202511"，然后点击「搜索」按钮
- 系统会显示该会计期间的所有总账记录

### 3.2 如何导出特定会计期间的总账数据？

- 先按会计期间筛选出需要的数据
- 然后点击「导出」按钮
- 系统会只导出筛选后的总账数据

### 3.3 期初余额和期末余额有什么关系？

- 期初余额是指会计期间开始时的账户余额
- 期末余额 = 期初余额 + 本期借方合计 - 本期贷方合计（根据余额方向不同可能有所调整）
- 期末余额方向由计算结果的正负决定

### 3.4 如何理解余额方向？

- 余额方向分为「借」和「贷」两种
- 资产类科目通常为借方余额
- 负债和所有者权益类科目通常为贷方余额
- 收入类科目通常为贷方余额
- 费用类科目通常为借方余额

## 4. 注意事项

- 总账数据是企业财务核算的基础，修改时需谨慎
- 确保输入的金额数据准确无误
- 定期备份总账数据，防止数据丢失
- 遵循企业的财务制度和会计准则进行操作
- 操作完成后及时核对数据，确保账账相符、账实相符
