# 采购-发货准备模块帮助文档

## 功能概述

采购-发货准备模块位于左侧菜单「ERP」→「采购」→「发货准备」，主要包含两大功能板块：

1. **成品包装**：用于管理成品的包装过程，包括选择仓库、时间范围、搜索SKU等操作，并生成包装配货单。
2. **批量组装**：用于管理产品的批量组装过程，包括处理组装单、查看组装状态、下载相关文档等操作。

## 操作指南

### 成品包装

1. **进入模块**：点击左侧菜单「ERP」→「采购」→「发货准备」，默认进入「成品包装」标签页。

2. **选择仓库**：在顶部的仓库选择器中选择要操作的仓库。

3. **设置时间范围**：使用日期选择器设置查询的时间范围，支持快速选择近7天、近一个月、近3个月。

4. **搜索SKU**：在搜索框中输入SKU或订单编号进行搜索。

5. **创建发货准备计划**：
   - 点击「创建发货准备」按钮，系统会自动创建一个发货准备运行单。
   - 在表格中选择需要包装的SKU，点击「添加到发货准备」按钮。

6. **生成包装配货单**：
   - 选择已添加到发货准备的SKU，点击「生成包装配货单」按钮。
   - 系统会生成PDF格式的包装配货单，包含产品信息、到货数量、库存信息、库位信息及耗材清单。

### 批量组装

1. **进入模块**：点击「批量组装」标签页。

2. **查询组装单**：
   - 选择仓库、组装单状态、搜索关键词等条件进行查询。
   - 支持按加工单编码、SKU、创建人等字段进行搜索。

3. **查看组装单详情**：
   - 点击组装单列表中的「详情」按钮，查看组装单的详细信息。
   - 包括主SKU、子组装件、耗材、加工数量等信息。

4. **下载组装单**：
   - 点击「下载」按钮，下载组装单的详细信息。
   - 支持下载PDF格式的组装配货单。

5. **批量操作**：
   - 选择多个组装单，点击「批量组装」按钮，系统会自动完成组装过程。
   - 组装完成后，组装单状态会更新为「已完成」。

6. **调库操作**：
   - 选择已完成的组装单，点击「调库」按钮，将组装好的产品调往指定仓库。

## 常见问题

1. **如何选择多个SKU进行包装？**
   - 在成品包装标签页的表格中，勾选需要包装的SKU，然后点击「添加到发货准备」按钮。

2. **如何生成包装配货单？**
   - 选择已添加到发货准备的SKU，点击「生成包装配货单」按钮，系统会自动生成PDF格式的配货单。

3. **如何查看组装单的状态？**
   - 在批量组装标签页的表格中，组装单状态会显示为「处理中」、「已完成」、「已终止」或「已作废」。

4. **如何处理组装过程中的异常？**
   - 如果组装过程中出现异常，可以点击「终止」按钮终止组装单，或联系系统管理员进行处理。

5. **如何查看组装单的详细信息？**
   - 点击组装单列表中的「详情」按钮，查看组装单的详细信息，包括主SKU、子组装件、耗材、加工数量等。

## 注意事项

1. **仓库选择**：操作前请确保选择了正确的仓库，不同仓库的库存和SKU可能不同。

2. **时间范围**：设置合适的时间范围，避免查询结果过多导致系统响应缓慢。

3. **组装单状态**：批量组装操作只能对状态为「处理中」的组装单进行。

4. **权限控制**：操作此模块需要具有相应的权限，请确保您有足够的权限进行操作。

5. **数据准确性**：操作前请仔细核对SKU、数量等信息，确保数据的准确性。

6. **系统性能**：当处理大量数据时，系统可能会响应较慢，请耐心等待。
