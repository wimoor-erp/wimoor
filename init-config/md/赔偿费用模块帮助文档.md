# 赔偿费用模块帮助文档

## 1. 模块概述

**赔偿费用模块**是一个专门用于查看和管理亚马逊FBA赔偿费用的工具。该模块通过整合亚马逊平台的赔偿数据，为用户提供详细的赔偿费用记录，帮助用户了解亚马逊对FBA库存损失或损坏的赔偿情况。

### 1.1 核心功能
- **赔偿费用查询**：按店铺、站点、日期范围等维度查询赔偿费用记录
- **详细数据展示**：展示赔偿费用的详细信息，包括商品信息、赔偿原因、金额等
- **数据导出**：支持导出赔偿费用数据为Excel文件
- **搜索功能**：支持按ASIN或平台SKU搜索赔偿记录

### 1.2 适用场景
- 查看亚马逊FBA赔偿费用明细
- 分析赔偿费用趋势和原因
- 核对赔偿金额是否正确
- 导出赔偿数据用于财务核算

## 2. 操作指南

### 2.1 进入模块
在系统导航菜单中，依次点击：**亚马逊** → **报表** → **赔偿费用**

### 2.2 界面布局

#### 2.2.1 筛选区域
- **店铺选择**：选择要查看的亚马逊店铺
- **日期选择**：选择赔偿费用的日期范围
- **搜索框**：输入ASIN或平台SKU进行搜索

#### 2.2.2 数据展示区域
- **表格展示**：以表格形式展示赔偿费用记录
- **商品信息**：显示商品图片、名称、SKU、ASIN等信息
- **费用时间**：显示赔偿批准时间和订单号
- **报销原因**：显示赔偿的具体原因和报销号
- **金额信息**：显示单个金额、总金额、币种
- **数量信息**：显示已偿还数量、已报销数量、已报数量总计

#### 2.2.3 操作按钮
- **导出**：导出当前筛选条件下的赔偿费用数据为Excel文件

### 2.3 主要操作流程

#### 2.3.1 查看赔偿费用
1. 选择要查看的店铺（可多选）
2. 选择日期范围（支持自定义日期范围）
3. 如需搜索特定商品，在搜索框中输入ASIN或平台SKU
4. 系统自动加载并显示符合条件的赔偿费用记录

#### 2.3.2 导出赔偿费用数据
1. 设置好筛选条件
2. 点击"导出"按钮
3. 等待系统生成Excel文件
4. 下载并保存Excel文件

## 3. 数据指标说明

### 3.1 商品信息
- **商品图片**：商品的主图
- **商品名称**：商品的标题
- **SKU**：平台SKU
- **ASIN**：亚马逊标准识别码

### 3.2 赔偿信息
- **费用时间**：赔偿批准的时间
- **订单号**：相关的亚马逊订单号
- **报销原因**：亚马逊赔偿的具体原因
- **报销号**：亚马逊系统生成的报销编号

### 3.3 金额信息
- **单个金额**：每个商品的赔偿金额
- **总金额**：该笔赔偿的总金额
- **币种**：赔偿金额的货币类型

### 3.4 数量信息
- **已偿还数量**：已通过现金形式偿还的商品数量
- **已报销数量**：已通过库存形式报销的商品数量
- **已报数量总计**：已报销的商品总数量

## 4. 常见问题

### 4.1 数据准确性问题
- **问题**：赔偿费用数据与亚马逊后台不符
  **解决方法**：系统会定期同步亚马逊的赔偿数据，若发现数据不符，请等待系统同步或联系技术支持

### 4.2 数据导出问题
- **问题**：导出的Excel文件为空或数据不完整
  **解决方法**：检查筛选条件是否正确，确保有符合条件的数据

### 4.3 搜索功能问题
- **问题**：搜索ASIN或SKU后没有结果
  **解决方法**：确保输入的ASIN或SKU格式正确，且在所选的日期范围内有赔偿记录

### 4.4 权限问题
- **问题**：无法查看某些店铺的赔偿费用数据
  **解决方法**：联系系统管理员，确认是否有相应的查看权限

## 5. 最佳实践

### 5.1 定期查看建议
- **定期检查**：建议每月查看一次赔偿费用记录，及时了解亚马逊的赔偿情况
- **对比分析**：与历史赔偿数据进行对比，分析赔偿原因和金额的变化趋势
- **异常监控**：关注大额赔偿或频繁出现的赔偿原因，及时发现和解决问题

### 5.2 数据利用建议
- **财务核算**：将赔偿费用数据纳入财务核算，准确计算实际成本
- **库存管理**：根据赔偿记录，优化库存管理策略，减少库存损失
- **供应商评估**：分析赔偿原因，评估供应商的产品质量

### 5.3 系统使用建议
- **合理筛选**：使用筛选功能，快速定位关注的赔偿记录
- **导出备份**：定期导出赔偿数据，作为数据备份和历史记录
- **组合查询**：结合店铺、日期和搜索功能，实现精准查询
