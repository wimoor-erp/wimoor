# 设置-角色权限模块帮助文档

## 1. 模块概述

角色权限模块是Wimoor系统中的核心管理模块之一，位于系统设置菜单下，主要用于管理员创建和管理角色，并为不同角色分配相应的菜单权限和功能权限。该模块采用Vue 3 + Element Plus开发，提供了直观、友好的权限管理界面，帮助管理员实现精细化的权限控制。

## 2. 功能结构

角色权限模块包含以下主要功能：

| 功能模块 | 子功能 | 描述 |
|---------|--------|------|
| 角色管理 | 角色列表 | 查看系统中所有角色 |
| 角色管理 | 添加角色 | 创建新的角色 |
| 角色管理 | 编辑角色 | 修改现有角色信息 |
| 角色管理 | 复制角色 | 基于现有角色创建新角色 |
| 角色管理 | 删除角色 | 删除不需要的角色 |
| 菜单权限管理 | 菜单分配 | 为角色分配可访问的菜单 |
| 菜单权限管理 | 权限继承 | 支持菜单权限的层级继承 |
| 功能权限管理 | 功能权限分配 | 为角色分配具体页面的功能权限 |
| 功能权限管理 | 字段权限分配 | 为角色分配字段查看权限 |

## 3. 操作指南

### 3.1 角色管理

#### 3.1.1 查看角色列表

1. **操作步骤**：
   - 进入角色权限管理页面
   - 左侧面板显示系统中所有角色列表
   - 可以通过搜索框筛选角色

2. **界面说明**：
   - 角色列表显示角色名称
   - 每个角色右侧有操作下拉菜单

#### 3.1.2 添加角色

1. **操作步骤**：
   - 点击左上角「添加角色」按钮
   - 在弹出的对话框中输入角色名称
   - 点击「确定」按钮保存

2. **注意事项**：
   - 角色名称不能为空
   - 角色名称不能重复

#### 3.1.3 编辑角色

1. **操作步骤**：
   - 在角色列表中，点击目标角色右侧的下拉菜单
   - 选择「编辑」选项
   - 在弹出的对话框中修改角色名称
   - 点击「确定」按钮保存修改

#### 3.1.4 复制角色

1. **操作步骤**：
   - 在角色列表中，点击目标角色右侧的下拉菜单
   - 选择「复制」选项
   - 在弹出的对话框中输入新角色名称
   - 点击「确定」按钮保存

2. **功能说明**：
   - 复制角色会同时复制原角色的所有权限设置

#### 3.1.5 删除角色

1. **操作步骤**：
   - 在角色列表中，点击目标角色右侧的下拉菜单
   - 选择「删除」选项
   - 在弹出的确认对话框中点击「确定」

2. **注意事项**：
   - 已被用户使用的角色无法删除
   - 删除角色前需先移除该角色下的所有用户

### 3.2 菜单权限管理

#### 3.2.1 分配菜单权限

1. **操作步骤**：
   - 在左侧角色列表中选择一个角色
   - 中间面板显示菜单权限树
   - 勾选需要分配给该角色的菜单
   - 系统自动保存权限设置

2. **权限说明**：
   - 勾选父菜单会自动勾选所有子菜单
   - 取消勾选父菜单会自动取消所有子菜单的勾选
   - 带指针图标的菜单表示包含功能权限

### 3.3 功能权限管理

#### 3.3.1 分配功能权限

1. **操作步骤**：
   - 在左侧角色列表中选择一个角色
   - 在中间菜单树中选择一个具体菜单
   - 右侧面板显示该菜单对应的功能权限
   - 勾选需要分配给该角色的功能权限
   - 系统自动保存权限设置

#### 3.3.2 分配字段权限

1. **操作步骤**：
   - 在左侧角色列表中选择一个角色
   - 在中间菜单树中选择一个具体菜单
   - 右侧面板的「字段查看权限」区域显示该菜单对应的字段权限
   - 勾选需要分配给该角色的字段权限
   - 系统自动保存权限设置

## 4. 权限管理最佳实践

### 4.1 角色设计原则

1. **基于职责设计**：根据用户的实际工作职责设计角色
2. **最小权限原则**：只分配完成工作所需的最小权限
3. **权限分层**：建立层次化的角色体系，如管理员、运营人员、客服人员等
4. **权限继承**：利用角色复制功能快速创建相似权限的角色

### 4.2 常见角色配置

| 角色名称 | 权限范围 | 适用场景 |
|---------|---------|----------|
| 系统管理员 | 所有权限 | 系统超级管理员 |
| 运营主管 | 大部分运营相关权限 | 运营部门负责人 |
| 运营专员 | 部分运营权限 | 普通运营人员 |
| 客服专员 | 客服相关权限 | 客服人员 |
| 财务专员 | 财务相关权限 | 财务人员 |

## 5. 常见问题

### 5.1 角色无法删除

**问题**：点击删除角色按钮后提示无法删除
**解决方案**：
- 检查该角色是否已被用户使用
- 先将使用该角色的用户移除该角色或修改为其他角色
- 再尝试删除角色

### 5.2 菜单权限不生效

**问题**：为角色分配了菜单权限，但用户登录后仍然无法看到菜单
**解决方案**：
- 检查是否正确选择了角色
- 检查菜单权限是否正确勾选
- 尝试刷新页面或重新登录系统

### 5.3 功能权限不生效

**问题**：为角色分配了功能权限，但用户登录后仍然无法使用相应功能
**解决方案**：
- 检查是否正确选择了角色
- 检查是否选择了正确的菜单
- 检查功能权限是否正确勾选
- 尝试刷新页面或重新登录系统

### 5.4 角色名称重复

**问题**：添加角色时提示角色名称重复
**解决方案**：
- 更换其他角色名称
- 检查系统中是否已存在同名角色
