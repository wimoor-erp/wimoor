# 账期SKU模块帮助文档

## 1. 模块概述

账期SKU模块是财务管理系统中用于查询、管理和分析账期内SKU级别的财务数据的功能模块。该模块提供了全面的账期SKU数据查询功能，支持按店铺、日期、负责人等多条件筛选，同时提供结账、重算、导出等功能，帮助用户清晰了解每个SKU的财务状况。

### 1.1 核心功能

- **多维度筛选**：支持按店铺、日期类型、日期范围、负责人等多条件筛选
- **币种切换**：支持在人民币(CNY)、美元(USD)、站点币种之间切换查看
- **关键词搜索**：支持搜索平台SKU
- **结账功能**：用于锁定汇率，确保财务数据的准确性
- **重算功能**：支持手动触发数据重算，确保数据的及时性
- **数据导出**：支持导出账期SKU数据
- **表格切换**：支持在水平表格和垂直表格之间切换查看

### 1.2 技术特点

- **前后端分离**：前端使用Vue 3 + Element Plus实现，后端基于Spring Boot构建
- **实时数据**：支持实时查询和分析账期SKU数据
- **多币种支持**：内置汇率转换功能，支持多币种切换查看
- **高性能**：优化大数据量下的查询和处理性能
- **用户友好**：直观的界面设计和操作流程

## 2. 功能说明

### 2.1 多维度筛选

**功能描述**：根据店铺、日期类型、日期范围、负责人等条件筛选账期SKU数据。

**操作步骤**：
1. 在页面顶部的店铺选择器中选择特定店铺
2. 在日期类型选择器中选择"结算日期"或"转账日期"
3. 在日期选择器中选择特定的日期范围
4. 在负责人选择器中选择特定负责人
5. 点击"查询"按钮

**说明**：
- 店铺选择器：支持选择特定店铺
- 日期类型：结算日期指账期的结算日期，转账日期指资金到账日期
- 日期范围：支持选择任意日期范围
- 负责人：支持选择特定的产品负责人

### 2.2 币种切换

**功能描述**：切换账期SKU数据的显示币种。

**币种选项**：
- **人民币**：以人民币显示账期SKU数据
- **美元**：以美元显示账期SKU数据
- **站点币种**：以各站点当地币种显示账期SKU数据

**操作步骤**：
1. 在页面顶部的币种选择器中选择需要的币种
2. 系统自动刷新账期SKU数据列表，显示转换后的金额

**说明**：
- 系统会自动进行汇率转换，确保数据准确性
- 不同币种之间的转换基于实时汇率

### 2.3 关键词搜索

**功能描述**：通过关键词搜索特定SKU的账期数据。

**操作步骤**：
1. 在页面顶部的搜索输入框中输入平台SKU
2. 系统自动刷新账期SKU数据列表，显示匹配的结果

**说明**：
- 搜索框支持模糊搜索
- 搜索结果会实时更新

### 2.4 结账功能

**功能描述**：用于锁定汇率，确保财务数据的准确性。

**操作步骤**：
1. 确保已设置好筛选条件
2. 点击页面顶部的"结账"按钮
3. 在弹出的结账对话框中确认操作
4. 系统开始处理结账操作
5. 操作完成后，系统会显示成功提示

**说明**：
- 结账功能会锁定当前筛选条件下的汇率
- 结账后的数据将保持汇率不变，确保财务数据的一致性
- 结账操作会清除相关缓存，确保数据的准确性

### 2.5 重算功能

**功能描述**：手动触发数据重算，确保数据的及时性。

**操作步骤**：
1. 点击页面顶部的"重算3"按钮
2. 在弹出的重算对话框中确认操作
3. 系统开始重新计算账期SKU数据
4. 计算完成后，系统会显示成功提示
5. 账期SKU数据列表会自动刷新

**说明**：
- 重算功能会重新计算所有相关的账期SKU数据
- 重算过程可能需要一定时间，请耐心等待
- 建议在数据异常或需要最新数据时使用此功能

### 2.6 数据导出

**功能描述**：导出账期SKU数据。

**操作步骤**：
1. 设置好需要导出的筛选条件
2. 点击页面顶部的"导出"按钮
3. 系统开始生成Excel文件并自动下载
4. 打开下载的Excel文件查看详细数据

**说明**：
- 导出的文件为Excel格式
- 导出的数据包含当前筛选条件下的所有账期SKU数据
- 导出文件包含SKU、销售额、费用、利润等详细信息

### 2.7 表格切换

**功能描述**：在水平表格和垂直表格之间切换查看。

**操作步骤**：
1. 在表格组件中点击表格切换按钮
2. 系统会自动切换表格显示方式
3. 表格数据会根据新的显示方式重新加载

**说明**：
- 水平表格：以水平方式展示账期SKU数据
- 垂直表格：以垂直方式展示账期SKU数据
- 两种表格展示方式的数据内容相同，只是布局不同

## 3. 操作指南

### 3.1 基本查询操作

**步骤1：进入账期SKU页面**
- 从系统菜单中选择"财务管理" > "账期SKU"

**步骤2：设置筛选条件**
- **店铺**：从店铺选择器中选择特定店铺
- **币种**：选择显示币种（人民币、美元、站点币种）
- **日期类型**：选择"结算日期"或"转账日期"
- **日期范围**：点击日期选择器，选择开始和结束日期
- **负责人**：从负责人选择器中选择特定负责人（可选）
- **关键词搜索**：在搜索输入框中输入平台SKU（可选）

**步骤3：执行查询**
- 点击"查询"按钮
- 系统会根据筛选条件刷新账期SKU数据列表

**步骤4：查看结果**
- 浏览账期SKU数据列表，查看详细信息
- 可根据需要切换表格显示方式

### 3.2 结账操作

**步骤1：准备工作**
- 确保已设置好需要结账的筛选条件

- 结账完成后，系统会显示成功提示

- 账期SKU数据列表会自动刷新

### 3.3 重算操作

**步骤1：执行重算操作**
- 点击页面顶部的"重算3"按钮
- 系统会弹出重算对话框
- 在对话框中确认操作
- 系统开始重新计算账期SKU数据

**步骤2：查看结果**
- 重算完成后，系统会显示成功提示
- 账期SKU数据列表会自动刷新
- 查看更新后的账期SKU数据

### 3.4 导出操作

**步骤1：设置导出条件**
- 确保已设置好需要导出的筛选条件
- 确认筛选条件下的数据是需要导出的数据

**步骤2：执行导出操作**
- 点击页面顶部的"导出"按钮
- 系统开始生成Excel文件
- 导出过程中，按钮会显示加载状态

**步骤3：查看导出结果**
- 导出完成后，系统会自动下载Excel文件

## 4. 常见问题

### 4.1 数据不显示或显示异常

**问题描述**：账期SKU数据不显示或显示异常。

**解决方案**：
- 检查筛选条件是否正确设置
- 尝试使用"重算3"功能刷新数据
- 检查日期范围是否设置正确
- 确认所选店铺是否有账期SKU数据
- 如果问题持续，联系系统管理员

### 4.2 币种转换不准确

**问题描述**：不同币种之间的转换结果与预期不符。

**解决方案**：
- 系统使用的是实时汇率，可能与您参考的汇率有所不同
- 确认选择的目标币种是否正确
- 检查汇率转换设置是否正确
- 如果差异较大，联系系统管理员检查汇率数据

### 4.3 导出数据失败

**问题描述**：点击"导出"按钮后，数据导出失败。

**解决方案**：
- 检查网络连接是否正常
- 确认筛选条件下是否有数据可导出
- 尝试缩小数据范围，减少导出的数据量
- 检查浏览器是否阻止了文件下载
- 如果问题持续，联系系统管理员

### 4.4 重算功能无反应

**问题描述**：点击"重算3"按钮后，系统没有反应。

**解决方案**：
- 检查网络连接是否正常
- 确认当前用户是否有重算权限
- 尝试刷新页面后再次操作
- 检查系统是否正在进行其他重算任务
- 如果问题持续，联系系统管理员

### 4.5 结账操作失败

**问题描述**：点击"结账"按钮后，结账操作失败。

**解决方案**：
- 检查筛选条件是否正确设置
- 确认筛选条件下是否有数据可结账
- 检查网络连接是否正常
- 尝试刷新页面后再次操作
- 如果问题持续，联系系统管理员

### 4.6 页面加载缓慢

**问题描述**：账期SKU页面加载缓慢，特别是在选择较大日期范围时。

**解决方案**：
- 缩小日期范围，减少查询的数据量
- 增加筛选条件，缩小查询范围
- 确保网络连接稳定
- 考虑在非高峰期使用系统
- 尝试使用"重算3"功能预处理数据

## 5. 最佳实践

### 5.1 查询技巧

1. **精确查询**：使用具体的店铺、日期范围和负责人进行精确查询
2. **分批查询**：对于较长时间范围的数据，建议分批查询
3. **合理筛选**：根据实际需求选择合适的筛选条件，避免不必要的数据加载
4. **币种选择**：分析国内成本使用人民币，分析国际市场使用美元
5. **日期类型**：分析账期使用"结算日期"，分析资金到账使用"转账日期"

### 5.2 操作建议

1. **定期检查**：定期检查账期SKU数据，确保财务数据的准确性
2. **及时结账**：在每个账期结束后及时进行结账操作，锁定汇率
3. **定期重算**：定期使用重算功能，确保数据的及时性
4. **数据备份**：定期导出账期SKU数据作为备份
5. **异常监控**：关注SKU级别的财务异常，及时发现问题

### 5.3 数据分析

1. **盈利能力分析**：分析不同SKU的盈利能力，找出高利润和低利润SKU
2. **成本分析**：分析SKU的成本构成，找出成本控制的关键点
3. **趋势分析**：分析SKU在不同账期的财务表现，识别趋势
4. **对比分析**：对比不同店铺、不同负责人的SKU财务表现
5. **异常识别**：通过数据波动识别异常的SKU财务数据

### 5.4 系统使用

1. **浏览器选择**：建议使用Chrome、Firefox等现代浏览器
2. **缓存清理**：定期清理浏览器缓存，确保系统正常运行
3. **网络环境**：确保在稳定的网络环境下使用系统
4. **权限管理**：合理设置用户权限，确保数据安全
5. **培训学习**：新用户建议先阅读本帮助文档，了解系统功能
