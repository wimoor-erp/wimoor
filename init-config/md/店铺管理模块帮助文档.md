# 店铺管理模块帮助文档

## 1. 模块概述

店铺管理模块是Wimoor系统中用于管理亚马逊店铺信息的功能模块，通过该模块用户可以添加、编辑、删除店铺信息，设置店铺排序，以及配置店铺相关的财务和海关信息。

## 2. 功能说明

### 2.1 核心功能

- **店铺管理**：支持添加、编辑、删除店铺信息
- **店铺排序**：支持通过拖拽方式调整店铺显示顺序
- **财务设置**：支持设置店铺是否为财务账套，配置公司名称和税号
- **海关信息**：支持配置海关注册编码和海关Dxpid
- **店铺列表**：展示已创建的店铺列表，包含店铺基本信息

### 2.2 高级功能

- **批量排序**：通过拖拽方式批量调整店铺的显示顺序
- **财务初始化**：支持初始化店铺的财务账套
- **权限控制**：基于用户权限管理店铺的访问和操作

## 3. 使用流程

### 3.1 添加店铺

1. **进入模块**：在系统菜单中找到"亚马逊" -> "店铺授权"模块
2. **点击新增**：点击页面左侧"店铺账号列表"区域的"新增"按钮
3. **填写信息**：
   - 输入店铺名称（必填）
   - 输入海关Dxpid（可选）
   - 输入海关注册编号（可选）
4. **保存店铺**：点击"确认"按钮保存店铺信息
5. **初始化财务账套**（可选）：
   - 保存店铺后，重新编辑店铺信息
   - 开启"是否财务账套"开关
   - 输入公司名称和税号
   - 开启"初始化财务账套"开关
   - 点击"确认"按钮完成初始化

### 3.2 编辑店铺

1. **进入模块**：在系统菜单中找到"亚马逊" -> "店铺授权"模块
2. **选择店铺**：在左侧店铺列表中找到需要编辑的店铺
3. **点击编辑**：点击店铺右侧的编辑图标
4. **修改信息**：修改店铺名称、海关信息或财务信息
5. **保存修改**：点击"确认"按钮保存修改

### 3.3 删除店铺

1. **进入模块**：在系统菜单中找到"亚马逊" -> "店铺授权"模块
2. **选择店铺**：在左侧店铺列表中找到需要删除的店铺
3. **点击删除**：点击店铺右侧的删除图标
4. **确认删除**：在弹出的确认对话框中点击"确认"按钮
5. **删除成功**：系统提示删除成功，店铺从列表中移除

### 3.4 调整店铺排序

1. **进入模块**：在系统菜单中找到"亚马逊" -> "店铺授权"模块
2. **点击排序**：点击左侧"店铺账号列表"区域的"排序"按钮
3. **拖拽排序**：在弹出的排序对话框中，通过拖拽方式调整店铺的显示顺序
4. **保存排序**：点击"确认"按钮保存排序结果
5. **排序成功**：系统提示排序成功，店铺列表按照新的顺序显示

## 4. 常见问题

### 4.1 无法删除店铺怎么办？

- **问题原因**：如果店铺下存在授权信息，系统会阻止删除操作
- **解决方法**：先删除店铺下的所有授权信息，然后再删除店铺

### 4.2 如何初始化财务账套？

- **操作步骤**：
  1. 先保存店铺基本信息
  2. 重新编辑店铺信息
  3. 开启"是否财务账套"开关
  4. 填写公司名称和税号
  5. 开启"初始化财务账套"开关
  6. 点击"确认"按钮完成初始化

### 4.3 店铺排序不生效怎么办？

- **问题原因**：可能是排序操作未成功保存
- **解决方法**：重新进入排序对话框，调整顺序后点击"确认"按钮保存

### 4.4 可以创建多少个店铺？

- **系统限制**：系统没有明确限制店铺数量，用户可以根据实际需求创建多个店铺
- **建议**：为了管理方便，建议根据实际的亚马逊店铺数量创建对应的店铺信息

## 5. 注意事项

### 5.1 操作注意事项

- **店铺名称**：店铺名称用于区分不同的店铺，建议使用与亚马逊店铺对应的名称
- **财务账套**：初始化财务账套后，系统会自动创建相关的财务科目和设置
- **海关信息**：如果需要使用系统的海关申报功能，建议填写完整的海关信息
- **店铺删除**：删除店铺前请确保已备份相关数据，删除后无法恢复

### 5.2 权限注意事项

- **管理员权限**：系统管理员可以查看和管理所有店铺
- **普通用户**：普通用户只能查看和管理所属公司的店铺
- **权限控制**：店铺的操作权限受系统角色权限控制

### 5.3 技术注意事项

- **浏览器兼容性**：建议使用Chrome、Firefox、Edge等现代浏览器
- **网络连接**：操作过程中请保持网络连接稳定
- **数据保存**：系统会自动保存操作数据，但建议在重要操作后确认数据保存成功
