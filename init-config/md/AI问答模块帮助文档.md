# AI问答模块帮助文档

## 1. 模块概述

AI问答模块是Wimoor系统中的一个智能工具，基于DeepSeek AI模型提供智能问答服务。该模块允许用户创建多个对话会话，与AI助手进行自然语言交互，获取问题解答、信息查询等服务。

**主要功能特点：**
- 多会话管理，支持创建和切换多个对话
- 基于DeepSeek AI模型的智能问答
- Markdown格式回答，支持代码高亮和格式化文本
- 对话历史记录和保存
- 常用搜索关键词推荐
- 支持深度搜索模式

**适用场景：**
- 技术问题咨询
- 业务信息查询
- 文档生成和编辑
- 数据分析和处理建议
- 日常工作助手

| 消息交互 | 消息发送 | 提交消息给AI助手 |
| 消息交互 | 消息展示 | 以Markdown格式展示AI回答 |

## 3. 操作指南

### 3.1 前端操作流程

#### 3.1.1 新建会话
1. 登录系统后，进入**系统管理** > **AI问答**模块
2. 在左侧会话列表中，点击**新建会话**按钮
3. 系统会创建一个新的空白会话，并显示初始欢迎消息
4. 在底部输入框中输入您的问题，按Enter键或点击**提交**按钮发送

#### 3.1.2 发送消息
1. 在底部输入框中输入您的问题或指令
2. 可以点击右侧的常用关键词，快速输入常见问题
3. 按Enter键或点击**提交**按钮发送消息
4. 等待AI助手生成回答，期间会显示加载状态
5. 回答生成完成后，会以Markdown格式显示在对话区域

#### 3.1.3 切换会话
1. 在左侧会话列表中，点击您想要查看的会话标题
2. 系统会切换到该会话，并显示完整的对话历史
3. 您可以继续在当前会话中与AI助手交互

#### 3.1.4 使用深度搜索模式
1. 在输入框下方，点击**深度搜索**按钮
2. 按钮变为蓝色，表示已切换到深度搜索模式
3. 在深度搜索模式下，AI助手会提供更详细、更深入的回答
4. 再次点击按钮可切换回普通模式

## 4. 常见问题与解决方案

### 4.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|----------|
| 消息发送失败 | 网络连接问题 | 检查网络连接，重试发送 |
| AI回答缓慢 | 服务器负载高或网络延迟 | 耐心等待，系统会自动处理 |
| 会话列表不显示 | 数据加载失败 | 刷新页面，重新进入模块 |
| 回答格式错乱 | Markdown渲染问题 | 刷新页面，或检查浏览器兼容性 |
| 深度搜索无效果 | 模式切换未生效 | 确认按钮状态为蓝色，表示已切换到深度搜索模式 |

### 4.2 最佳实践

1. **问题描述清晰**：尽量详细描述您的问题，提供足够的上下文信息
2. **会话分类**：为不同类型的问题创建不同的会话，便于后续查找
3. **关键词使用**：利用右侧的常用关键词，快速发起常见问题的查询
4. **深度搜索**：对于复杂问题，使用深度搜索模式获取更详细的回答
5. **历史记录**：系统会自动保存对话历史，可随时回顾之前的对话内容
6. **格式要求**：如需特定格式的回答，明确告知AI助手（如"请以列表形式回答"）

## 5. 注意事项

1. **内容安全**：请勿在对话中输入敏感信息，如密码、银行卡号等
2. **使用限制**：AI模型有一定的使用限制，避免发送过于频繁的请求
3. **回答准确性**：AI回答仅供参考，重要信息请自行核实
4. **网络要求**：需要稳定的网络连接，以确保与AI服务的正常通信
5. **浏览器兼容性**：建议使用Chrome、Firefox等现代浏览器，获得最佳体验
6. **数据存储**：对话历史会保存在系统中，请合理使用存储空间
