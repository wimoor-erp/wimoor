# 设置-透明计划授权模块帮助文档

## 1. 模块概述

设置-透明计划授权模块是Wimoor系统中用于管理亚马逊透明计划（Transparency）授权信息的功能模块，通过该模块用户可以添加、编辑、删除透明计划的授权信息，实现与亚马逊透明计划服务的对接，用于后续的透明计划编码管理和产品验证。

## 2. 功能说明

### 2.1 核心功能

- **授权管理**：支持添加、编辑、删除透明计划授权信息
- **授权信息配置**：支持配置Client Id和Client Secret等授权信息
- **授权列表**：展示已创建的授权信息列表，包含名称、Client Id和创建时间
- **授权搜索**：支持根据名称搜索授权信息

### 2.2 高级功能

- **令牌管理**：自动管理透明计划API的访问令牌，包括令牌刷新和过期处理
- **权限控制**：基于用户权限管理授权信息的访问和操作

## 3. 使用流程

### 3.1 添加授权信息

1. **进入模块**：在系统菜单中找到"亚马逊" -> "透明计划" -> "授权管理"模块
2. **点击新增**：点击页面左上角的"新增"按钮
3. **填写信息**：
   - 输入授权名称（必填）
   - 输入Client Id（必填）
   - 输入Client Secret（必填）
4. **保存设置**：点击"确认"按钮保存授权信息

### 3.2 编辑授权信息

1. **进入模块**：在系统菜单中找到"亚马逊" -> "透明计划" -> "授权管理"模块
2. **选择授权**：在授权列表中找到需要编辑的授权信息
3. **点击编辑**：点击授权信息右侧的"编辑"按钮
4. **修改信息**：在弹出的编辑对话框中修改授权信息
5. **保存修改**：点击"确认"按钮保存修改

### 3.3 删除授权信息

1. **进入模块**：在系统菜单中找到"亚马逊" -> "透明计划" -> "授权管理"模块
2. **选择授权**：在授权列表中找到需要删除的授权信息
3. **点击删除**：点击授权信息右侧的"删除"按钮
4. **确认删除**：在弹出的确认对话框中点击"确认"按钮

### 3.4 搜索授权信息

1. **进入模块**：在系统菜单中找到"亚马逊" -> "透明计划" -> "授权管理"模块
2. **输入关键词**：在页面顶部的搜索输入框中输入授权名称关键词
3. **点击搜索**：点击搜索输入框右侧的搜索按钮
4. **查看结果**：系统会显示匹配的授权信息列表

## 4. 常见问题

### 4.1 无法添加授权信息怎么办？

- **问题原因**：可能是必填字段未填写完整，或Client Id和Client Secret不正确
- **解决方法**：检查所有必填字段是否已填写，确保Client Id和Client Secret的准确性

### 4.2 如何获取Client Id和Client Secret？

- **获取途径**：Client Id和Client Secret需要在亚马逊透明计划开发者平台申请
- **申请流程**：登录亚马逊透明计划开发者平台，创建应用程序，获取Client Id和Client Secret

### 4.3 授权信息过期怎么办？

- **系统处理**：系统会自动管理访问令牌的刷新，当令牌接近过期时会自动刷新
- **手动处理**：如果授权信息长时间未使用，可能需要重新编辑并保存授权信息，触发令牌刷新

### 4.4 可以添加多少个授权信息？

- **系统限制**：系统没有明确限制授权信息数量，用户可以根据实际需求添加多个授权信息
- **建议**：为了管理方便，建议只添加实际需要使用的授权信息

## 5. 注意事项

### 5.1 操作注意事项

- **授权信息安全**：Client Id和Client Secret属于敏感信息，请妥善保管
- **信息准确性**：确保填写的Client Id和Client Secret与亚马逊透明计划开发者平台的信息一致
- **定期检查**：定期检查授权信息的状态，确保授权有效

### 5.2 技术注意事项

- **网络连接**：确保系统服务器可以正常访问亚马逊透明计划API的网络地址
- **权限设置**：确保Client Id和Client Secret具有足够的API访问权限
- **令牌管理**：系统会自动管理令牌的刷新，但建议定期检查令牌状态

### 5.3 安全注意事项

- **授权信息保护**：不要在公共场合或不安全的网络环境下配置授权信息
- **定期更新**：定期更新授权信息，特别是当员工离职或权限变更时
- **访问控制**：限制授权信息的访问权限，只授予需要的人员
