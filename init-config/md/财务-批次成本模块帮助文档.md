# 财务-批次成本模块帮助文档

## 1. 模块概述

财务-批次成本模块是一个专为财务人员设计的工具，用于管理和分析亚马逊FBA发货的批次成本。通过本模块，财务人员可以：

- 查看FBA发货的批次成本明细
- 重置和调整批次成本
- 修改接收日期
- 重新计算运费分摊
- 查看扣减明细
- 导出批次成本数据

## 2. 功能说明

### 2.1 批次成本查询

**功能描述**：查询FBA发货的批次成本数据，包括采购成本和头程成本。

**操作步骤**：
1. 进入批次成本页面
2. 选择店铺分组和国家
3. 选择日期范围
4. 选择状态（未开始/扣减中/已扣完）
5. 输入搜索关键词（平台SKU/单据号/货件号）
6. 点击查询按钮查看批次成本数据

**数据说明**：
- **平台SKU**：亚马逊平台上的SKU
- **发货日期**：FBA发货的日期
- **接收日期**：亚马逊仓库接收的日期
- **状态**：批次成本的状态（未开始/扣减中/已扣完）
- **单据号**：发货单据编码
- **货件号**：亚马逊货件号
- **发出数量**：发货数量
- **接收数量**：亚马逊实际接收的数量
- **扣除数量**：已扣减的数量
- **剩余量**：剩余未扣减的数量
- **采购成本**：包括总成本和单件成本
- **头程成本**：包括总成本和单件成本

### 2.2 成本重置

**功能描述**：重置批次成本，包括采购成本和头程成本。

**操作步骤**：
1. 在批次成本列表中选择需要重置成本的记录
2. 点击"重置成本"按钮
3. 在弹出的对话框中输入新的成本数据
4. 点击"确认"按钮完成重置

### 2.3 接收日期修改

**功能描述**：修改FBA发货的接收日期。

**操作步骤**：
1. 在批次成本列表中选择需要修改接收日期的记录
2. 点击"修改接收日期"按钮
3. 在弹出的对话框中选择修改方式：
   - 具体日期：直接选择新的接收日期
   - 增加天数：在当前接收日期基础上增加指定天数
4. 点击"确认"按钮完成修改

### 2.4 运费重算

**功能描述**：重新计算货件的运费分摊。

**操作步骤**：
1. 在批次成本列表中找到需要重算运费的记录
2. 点击货件号旁边的"重算运费"按钮
3. 系统会自动重新计算并更新运费数据

### 2.5 扣减明细查看

**功能描述**：查看批次成本的扣减明细，包括订单信息、发货数量、售价等。

**操作步骤**：
1. 在批次成本列表中找到需要查看明细的记录
2. 点击"扣减明细"按钮
3. 在弹出的对话框中查看详细的扣减记录

### 2.6 数据导出

**功能描述**：导出批次成本数据为Excel文件。

**操作步骤**：
1. 设置查询条件，筛选需要导出的数据
2. 点击"导出"按钮
3. 系统会生成并下载Excel格式的批次成本报表

## 3. 操作指南

### 3.1 基本操作流程

1. **进入模块**：从左侧菜单导航到"亚马逊 > 批次成本"
2. **设置查询条件**：
   - 选择店铺分组和国家
   - 选择日期范围
   - 选择状态
   - 输入搜索关键词
3. **查看批次成本**：系统会显示符合条件的批次成本数据
4. **执行操作**：
   - 重置成本：选择记录，点击"重置成本"按钮
   - 修改接收日期：选择记录，点击"修改接收日期"按钮
   - 重算运费：点击货件号旁边的"重算运费"按钮
   - 查看扣减明细：点击"扣减明细"按钮
   - 导出数据：点击"导出"按钮

### 3.2 常见问题及解决方法

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 无法重置成本 | 未选择数据行 | 先选择需要重置成本的记录，再点击"重置成本"按钮 |
| 无法修改接收日期 | 未选择数据行 | 先选择需要修改接收日期的记录，再点击"修改接收日期"按钮 |
| 运费重算失败 | 货件信息不完整 | 确保货件信息完整，特别是重量信息 |
| 扣减明细为空 | 该批次尚未开始扣减 | 等待系统开始扣减后再查看明细 |
| 导出数据为空 | 查询条件设置不当 | 调整查询条件，确保有符合条件的数据 |

## 4. 注意事项

1. **数据准确性**：批次成本数据直接影响产品的成本核算，修改成本时请确保数据的准确性。

2. **接收日期影响**：修改接收日期会影响成本的计算时间，进而影响财务报表，请谨慎操作。

3. **运费分摊**：运费重算会根据货件的重量重新计算运费分摊，确保货件的重量信息准确。

4. **权限控制**：只有具有财务权限的用户才能访问本模块，请确保您的账号具有相应权限。

5. **数据安全**：本模块包含敏感的财务数据，请妥善保管您的账号和密码，避免数据泄露。

## 5. 术语解释

- **批次成本**：指一批FBA发货的总成本，包括采购成本和头程成本。
- **头程成本**：指将货物从国内运输到亚马逊仓库的费用。
- **采购成本**：指购买产品的成本。
- **扣减**：指随着产品销售，逐渐将批次成本分摊到销售成本中。
- **未开始**：批次成本尚未开始扣减。
- **扣减中**：批次成本正在扣减过程中。
- **已扣完**：批次成本已全部扣减完成。
- **货件号**：亚马逊系统生成的FBA货件编号。
