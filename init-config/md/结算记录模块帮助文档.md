# 结算记录模块帮助文档

## 1. 模块概述

结算记录模块是财务管理系统中用于查询、管理和分析亚马逊平台结算记录的功能模块。该模块提供了全面的结算记录查询功能，支持按结算状态、店铺、国家、日期等多条件筛选，同时提供汇总分析、明细下载等功能，帮助用户清晰了解店铺的回款情况。

### 1.1 核心功能

- **结算状态筛选**：支持筛选已结算和未结算的记录
- **多维度查询**：支持按店铺、国家、结算日期、转账日期等多维度筛选
- **币种切换**：支持在原币种、CNY、USD之间切换查看
- **数据汇总**：提供未结算金额、已结算金额、总金额等汇总数据
- **账期汇总**：支持校验账期汇总和强制重算功能
- **明细下载**：支持下载单个账期的明细和导出所有账期数据
- **状态可视化**：清晰展示结算记录的不同状态（未转账、转账失败、转账中、已转账）

### 1.2 技术特点

- **前后端分离**：前端使用Vue 3 + Element Plus实现，后端基于Spring Boot构建
- **实时数据**：支持实时查询和汇总结算数据
- **多币种支持**：内置汇率转换功能，支持多币种切换查看
- **高性能**：优化大数据量下的查询和处理性能
- **用户友好**：直观的界面设计和操作流程

## 2. 功能说明

### 2.1 结算状态筛选

**功能描述**：筛选已结算或未结算的记录。

**操作步骤**：
1. 在页面顶部的单选按钮组中选择"已结算"或"未结算"
2. 系统自动刷新结算记录列表

**说明**：
- 已结算：显示已经完成转账的结算记录
- 未结算：显示尚未完成转账的结算记录

### 2.2 多维度查询

**功能描述**：根据店铺、国家、日期等条件查询结算记录。

**查询条件**：
- **店铺**：通过店铺选择器选择特定店铺
- **国家**：选择特定国家/地区
- **日期类型**：选择按"结算日期"或"转账日期"筛选
- **日期范围**：选择特定的日期范围

**操作步骤**：
1. 选择店铺和国家（可选）
2. 选择日期类型（可选）
3. 选择日期范围
4. 系统自动刷新结算记录列表

### 2.3 币种切换

**功能描述**：切换结算记录的显示币种。

**币种选项**：
- **原币种**：显示结算记录的原始币种
- **CNY**：转换为人民币显示
- **USD**：转换为美元显示

**操作步骤**：
1. 在币种下拉菜单中选择需要的币种
2. 系统自动刷新结算记录列表，显示转换后的金额

**说明**：
- 选择"原币种"时，需要先选择特定国家，系统会自动使用该国家的币种
- 系统会自动进行汇率转换，显示转换后的金额

### 2.4 数据汇总

**功能描述**：显示结算记录的汇总数据。

**汇总数据**：
- **未结算金额**：尚未完成转账的金额
- **已结算金额**：已经完成转账的金额
- **总金额**：未结算金额和已结算金额的总和
- **转账失败金额**：转账失败的金额

**显示位置**：页面中部的卡片区域，实时更新

### 2.5 账期汇总

**功能描述**：对账期汇总数据进行校验和重算。

**操作步骤**：
1. 确保当前筛选条件为"已结算"
2. 点击"汇总"按钮，在下拉菜单中选择操作类型
3. **校验账期汇总**：系统自动校验账期汇总数据的准确性
4. **强制重算**：系统重新计算账期汇总数据

**说明**：
- 这些功能仅对"已结算"状态的记录有效
- 操作完成后，系统会自动刷新数据

### 2.6 明细下载

**功能描述**：下载结算记录的明细数据。

**下载选项**：
- **下载明细**：下载单个账期的详细记录
- **导出所有账期**：导出当前筛选条件下的所有账期数据

**操作步骤**：
1. **下载明细**：在结算记录列表中，找到需要下载的账期，点击"下载明细"按钮
2. **导出所有账期**：点击页面顶部的"导出所有账期"按钮，系统会提示确认，点击"确认"后开始下载

**说明**：
- 下载的文件为Excel格式
- 导出所有账期时，系统会限制最大导出数量为300个账期

### 2.7 状态可视化

**功能描述**：通过标签颜色直观展示结算记录的不同状态。

**状态说明**：
- **未转账**：黄色标签，表示结算周期已结束但尚未开始转账
- **转账失败**：红色标签，表示转账过程中出现失败
- **转账中**：黄色标签，表示转账正在进行中
- **已转账**：绿色标签，表示转账已成功完成

**显示位置**：结算记录列表的"状态"列

## 3. 操作指南

### 3.1 基本查询操作

**步骤1：进入结算记录页面**
- 从系统菜单中选择"财务管理" > "结算记录"

**步骤2：设置筛选条件**
- **结算状态**：选择"已结算"或"未结算"
- **店铺和国家**：从下拉菜单中选择特定店铺和国家（可选）
- **币种**：选择显示币种（原币种、CNY、USD）
- **日期类型**：选择按"结算日期"或"转账日期"筛选（可选）
- **日期范围**：点击日期选择器，选择开始和结束日期

**步骤3：查看结果**
- 系统自动根据筛选条件刷新结算记录列表
- 查看页面中部的汇总数据卡片
- 浏览结算记录列表，查看详细信息

### 3.2 账期汇总操作

**步骤1：准备工作**
- 确保当前筛选条件设置为"已结算"
- 选择需要处理的日期范围

**步骤2：执行汇总操作**
- 点击页面顶部的"汇总"按钮
- 在下拉菜单中选择"校验账期汇总"或"强制重算"
- 系统开始处理，显示加载状态

**步骤3：查看结果**
- 操作完成后，系统会显示成功提示
- 汇总数据卡片会自动更新
- 结算记录列表会自动刷新

### 3.3 明细下载操作

**下载单个账期明细**
1. 在结算记录列表中找到需要下载的账期
2. 点击该账期对应的"下载明细"按钮
3. 系统开始生成Excel文件并自动下载
4. 打开下载的Excel文件查看详细记录

**导出所有账期**
1. 设置好需要导出的筛选条件
2. 点击页面顶部的"导出所有账期"按钮
3. 在弹出的确认对话框中点击"确认"
4. 系统开始生成Excel文件并自动下载
5. 打开下载的Excel文件查看所有账期数据

### 3.4 状态查看操作

**查看转账状态**
1. 在结算记录列表中查看"状态"列
2. 根据标签颜色和文字判断转账状态
3. 同时可查看"转账状态"列，了解更详细的转账情况

**查看结算周期**
1. 在结算记录列表中查看"结算周期"列
2. 查看每个账期的开始和结束日期
3. 了解每个结算周期的时间范围

## 4. 常见问题

### 4.1 无法选择原币种

**问题描述**：选择"原币种"时提示"请选择对应的国家"

**解决方案**：
- 选择"原币种"前，需要先在店铺选择器中选择特定的国家
- 系统需要知道具体的国家才能确定使用哪种原币种

### 4.2 导出所有账期失败

**问题描述**：点击"导出所有账期"按钮后提示"最大导出账期数量不能超过300个"

**解决方案**：
- 缩小日期范围，减少导出的账期数量
- 分批导出不同时间段的账期数据
- 考虑使用更具体的筛选条件，减少符合条件的账期数量

### 4.3 账期汇总操作无反应

**问题描述**：点击"校验账期汇总"或"强制重算"按钮后没有反应

**解决方案**：
- 确保当前筛选条件设置为"已结算"
- 检查网络连接是否正常
- 确认选择了有效的日期范围
- 如果问题持续，联系系统管理员

### 4.4 币种转换不准确

**问题描述**：转换后的金额与预期不符

**解决方案**：
- 系统使用的是实时汇率，可能与您参考的汇率有所不同
- 确认选择的目标币种是否正确
- 如果差异较大，联系系统管理员检查汇率设置

### 4.5 页面加载缓慢

**问题描述**：结算记录页面加载缓慢，特别是在选择较大日期范围时

**解决方案**：
- 缩小日期范围，减少查询的数据量
- 增加筛选条件，缩小查询范围
- 确保网络连接稳定
- 考虑在非高峰期使用系统

## 5. 最佳实践

### 5.1 查询技巧

1. **精确查询**：使用具体的店铺、国家和日期范围进行精确查询
2. **分批查询**：对于较长时间范围的数据，建议分批查询
3. **合理筛选**：根据实际需求选择合适的筛选条件，避免不必要的数据加载
4. **币种选择**：根据分析需求选择合适的币种，CNY便于国内财务分析，USD便于国际对比
5. **日期类型**：分析结算周期使用"结算日期"，分析资金到账使用"转账日期"

### 5.2 操作建议

1. **定期检查**：定期检查结算记录，确保回款正常
2. **及时处理**：发现转账失败的记录及时处理
3. **汇总分析**：定期使用账期汇总功能，确保数据准确性
4. **数据备份**：定期导出结算数据作为备份
5. **异常监控**：关注未结算金额的变化，及时发现异常情况

### 5.3 数据分析

1. **趋势分析**：分析不同时期的结算金额变化趋势
2. **对比分析**：对比不同店铺、不同国家的结算情况
3. **效率分析**：分析结算周期和转账时间，评估资金周转效率
4. **异常识别**：通过汇总数据快速识别异常金额和异常状态
5. **预测分析**：基于历史数据预测未来的回款情况

### 5.4 系统使用

1. **浏览器选择**：建议使用Chrome、Firefox等现代浏览器
2. **缓存清理**：定期清理浏览器缓存，确保系统正常运行
3. **网络环境**：确保在稳定的网络环境下使用系统
4. **权限管理**：合理设置用户权限，确保数据安全
5. **培训学习**：新用户建议先阅读本帮助文档，了解系统功能
