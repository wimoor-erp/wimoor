# 物流报价设置页面帮助文档

## 1. 页面概述

`setting.vue` 是Wimoor ERP系统中物流模块下的报价设置页面，主要用于管理询价商和物流供应商信息。该页面分为左右两个主要功能区域：左侧为询价商管理，右侧为物流供应商管理。

## 2. 功能模块详解

### 2.1 询价商管理

询价商管理模块位于页面左侧，主要用于管理当前系统与询价商的绑定关系以及询价商信息。

#### 2.1.1 状态显示
- **已绑定**：显示当前已绑定的询价商令牌信息
- **未绑定**：提示用户需要绑定询价商令牌

#### 2.1.2 令牌管理
- **绑定令牌**：输入令牌和别名，点击"绑定"按钮完成绑定
- **复制令牌**：点击复制图标可复制当前绑定的令牌
- **解绑令牌**：点击"解绑"按钮解除当前绑定关系（需确认操作）

#### 2.1.3 询价商信息设置
在"高级"折叠面板中，可设置或修改询价商详细信息：
- 名称：询价商公司名称
- 地址：询价商公司地址
- 手机号：联系电话
- 联系人：对接人姓名

### 2.2 物流供应商管理

物流供应商管理模块位于页面右侧，主要用于管理与当前询价商关联的物流供应商列表。

#### 2.2.1 供应商列表
列表显示以下信息：
- 名称：物流供应商名称
- 联系人：对接人姓名及电话
- 链接：供应商页面链接（可点击访问或复制）
- 创建时间：供应商信息创建时间

#### 2.2.2 供应商操作
- **新增**：点击"新增"按钮打开创建对话框
- **编辑**：点击"编辑"按钮修改供应商信息
- **删除**：点击"删除"按钮移除供应商（需确认操作）

## 3. 操作步骤

### 3.1 绑定询价商令牌
1. 在左侧"询价商管理"区域，输入令牌和别名
2. 点击"绑定"按钮
3. 系统验证令牌有效性并完成绑定

### 3.2 设置询价商信息
1. 确保已绑定询价商令牌
2. 展开"高级"折叠面板
3. 点击"修改"按钮进入编辑模式
4. 填写或修改询价商信息（名称、地址、手机号、联系人）
5. 点击"保存"按钮完成设置

### 3.3 新增物流供应商
1. 确保已绑定询价商令牌
2. 在右侧"物流供应商管理"区域，点击"新增"按钮
3. 在弹出的创建对话框中填写供应商信息
4. 点击"保存"按钮完成新增

### 3.4 编辑物流供应商
1. 在供应商列表中找到需要修改的供应商
2. 点击"编辑"按钮
3. 在弹出的编辑对话框中修改信息
4. 点击"保存"按钮完成修改

### 3.5 删除物流供应商
1. 在供应商列表中找到需要删除的供应商
2. 点击"删除"按钮
3. 在确认对话框中点击"确定"按钮
4. 系统删除该供应商信息

## 4. 数据交互

### 4.1 API接口
- `orderApi.addBuyer()`：添加或更新询价商信息
- `orderApi.getBuyer()`：获取询价商信息
- `thirdpartyApi.saveQuoteToken()`：保存询价商令牌
- `thirdpartyApi.unbindQuoteToken()`：解绑询价商令牌
- `thirdpartyApi.getQuoteToken()`：获取当前绑定的令牌信息
- `supplierApi.listsupplier()`：获取物流供应商列表
- `supplierApi.deletesupplier()`：删除物流供应商

### 4.2 数据流程
1. 页面加载时，调用`loadToken()`获取当前绑定的令牌信息
2. 如果已绑定令牌，调用`loadBuyer()`和`loadSupplier()`获取询价商和供应商数据
3. 绑定/解绑令牌时，更新本地状态并重新加载数据
4. 新增/编辑/删除供应商时，调用相应API并刷新列表

## 5. 组件依赖

- **Element Plus**：UI组件库，提供表格、表单、按钮等组件
- **icon-park**：图标库，提供各种功能图标
- **CreateDialog**：子组件，用于创建和编辑供应商信息
- **copy_text.js**：工具函数，用于复制文本
- **dateFormat/dateTimesFormat**：日期格式化工具

## 6. 常见问题

### 6.1 绑定令牌失败
- 检查令牌是否正确
- 确认网络连接正常
- 联系询价商确认令牌有效性

### 6.2 无法看到供应商列表
- 确认已成功绑定询价商令牌
- 刷新页面重试
- 检查网络连接

### 6.3 无法删除供应商
- 确认当前用户有删除权限
- 检查供应商是否被其他业务引用

## 7. 技术说明

- **页面路径**：`wimoor-ui/src/views/erp/shipv2/quote/setting.vue`
- **主要技术栈**：Vue 3 + Element Plus
- **状态管理**：使用Vue 3的Composition API进行状态管理
- **响应式设计**：适配不同屏幕尺寸
