# 广告管理模块使用指南

## 1. 功能概述

广告管理模块是Wimoor系统中用于管理亚马逊平台广告活动的专业工具。该模块支持SP（ Sponsored Products ）、SB（ Sponsored Brands ）、SD（ Sponsored Display ）三种广告类型的全生命周期管理，包括广告活动创建、广告组设置、商品广告管理、关键词管理、定向管理等功能，帮助用户高效管理广告投放，优化广告效果，提升投资回报率。

### 1.1 模块价值

- **全类型支持**：支持SP、SB、SD三种亚马逊广告类型，满足不同推广需求
- **全流程管理**：从广告活动创建到关键词优化，实现广告全生命周期管理
- **数据驱动**：提供丰富的广告数据报表和分析工具，支持数据驱动决策
- **批量操作**：支持批量创建、修改、暂停广告，提高管理效率
- **智能优化**：提供出价建议、预算调整等智能优化功能，提升广告效果
- **权限控制**：完善的权限管理，确保广告操作安全可控

### 1.2 适用场景

- **新品推广**：为新品创建广告活动，快速获得曝光和流量
- **爆款打造**：通过关键词和定向优化，提升爆款商品的转化率
- **库存清理**：针对库存过多的商品，创建促销广告活动
- **品牌宣传**：使用SB广告类型，提升品牌知名度
- **竞品拦截**：通过SD广告定向竞品，拦截竞争对手流量
- **日常优化**：定期查看广告数据，优化关键词和出价策略

## 2. 页面访问

在系统左侧菜单栏中，依次点击：

**亚马逊** → **广告** → **广告管理**

### 2.1 权限说明

访问该模块需要具备以下权限：
- 亚马逊广告管理权限
- 对应店铺的广告操作权限

### 2.2 页面加载时间

- 首次加载时间：约5-10秒（取决于网络速度和广告数量）
- 广告活动切换后加载时间：约2-5秒
- 筛选条件变更后加载时间：约1-3秒

## 3. 界面介绍

### 3.1 主界面布局

广告管理模块主界面采用左右分栏布局，包含以下主要部分：

1. **左侧广告树**：位于页面左侧，展示广告账户、广告活动、广告组的层级结构
   - **账户节点**：显示亚马逊广告账户信息
   - **广告活动节点**：显示广告活动名称和状态
   - **广告组节点**：显示广告组名称和状态

2. **右侧操作区域**：位于页面右侧，占页面主要区域，包含以下内容：
   - **面包屑导航**：显示当前操作的层级路径，支持快速返回上级
   - **标签页导航**：根据广告类型和操作对象不同，显示不同的标签页
   - **数据表格**：展示广告活动、广告组、商品广告、关键词等数据
   - **操作按钮**：提供创建、编辑、暂停、删除等操作按钮
   - **数据图表**：展示广告数据趋势和分析

### 3.2 广告类型说明

| 广告类型 | 英文名称 | 描述 | 适用场景 |
|---------|---------|------|----------|
| SP广告 | Sponsored Products | 针对单个商品的广告，出现在搜索结果和商品详情页 | 推广具体商品，提升销量 |
| SB广告 | Sponsored Brands | 品牌广告，包含品牌 logo、自定义标题和多个商品 | 提升品牌知名度，推广品牌店铺 |
| SD广告 | Sponsored Display | 展示型广告，可定向特定商品、兴趣或受众 | 重定向浏览过的用户，拦截竞品流量 |

### 3.3 核心标签页

| 标签页名称 | 功能描述 | 适用广告类型 |
|---------|---------|----------|
| 广告活动 | 管理广告活动的创建、编辑、暂停等操作 | SP、SB、SD |
| 广告组 | 管理广告组的设置、预算、出价等 | SP、SB、SD |
| 商品 | 管理商品广告的状态、出价等 | SP、SD |
| 广告 | 管理品牌广告的创意、内容等 | SB |
| 关键词 | 管理关键词的添加、删除、出价调整等 | SP、SB |
| 用户搜索词 | 查看用户搜索词数据，优化关键词策略 | SP、SB |
| 商品投放 | 管理商品定向，包括投放和否定投放 | SP、SD |
| 用户投放搜索 | 查看用户投放数据，优化定向策略 | SP、SD |
| 广告组-否定关键词 | 管理广告组级别的否定关键词 | SP、SB |
| 活动-否定关键词 | 管理活动级别的否定关键词 | SP |
| 广告组-否定投放 | 管理广告组级别的否定商品投放 | SP、SD |
| 活动-否定投放 | 管理活动级别的否定商品投放 | SP |
| 商品购买 | 查看商品购买数据，分析广告效果 | SP、SB、SD |

## 4. 操作指南

### 4.1 广告活动管理

#### 4.1.1 创建广告活动

1. **选择广告类型**：在左侧广告树中选择对应的广告账户，点击右键选择"创建广告活动"
2. **填写基本信息**：
   - 广告活动名称：输入广告活动名称（建议包含商品名称和日期）
   - 广告类型：选择SP、SB或SD
   - 投放类型：选择自动投放或手动投放
   - 每日预算：设置广告活动的每日预算
   - 开始日期：设置广告活动的开始日期
   - 结束日期：可选，设置广告活动的结束日期
3. **设置投放策略**：
   - 出价策略：选择动态出价策略（仅降低、提高和降低、固定）
   - 投放位置：设置搜索结果顶部、商品详情页等位置的出价调整
4. **保存并创建**：点击"保存"按钮，系统会自动同步到亚马逊后台

#### 4.1.2 编辑广告活动

1. **选择广告活动**：在左侧广告树中选择要编辑的广告活动
2. **点击编辑按钮**：在右侧操作区域点击"编辑"按钮
3. **修改信息**：修改广告活动名称、预算、状态等信息
4. **保存修改**：点击"保存"按钮，系统会自动同步到亚马逊后台

#### 4.1.3 暂停/启用广告活动

1. **选择广告活动**：在左侧广告树中选择要操作的广告活动
2. **点击状态按钮**：在右侧操作区域点击"暂停"或"启用"按钮
3. **确认操作**：在弹出的确认对话框中点击"确定"

### 4.2 广告组管理

#### 4.2.1 创建广告组

1. **选择广告活动**：在左侧广告树中选择要添加广告组的广告活动
2. **点击创建广告组**：在右侧操作区域点击"创建广告组"按钮
3. **填写广告组信息**：
   - 广告组名称：输入广告组名称
   - 默认出价：设置广告组的默认出价
   - 每日预算：设置广告组的每日预算（可选）
4. **保存并创建**：点击"保存"按钮，系统会自动同步到亚马逊后台

#### 4.2.2 编辑广告组

1. **选择广告组**：在左侧广告树中选择要编辑的广告组
2. **点击编辑按钮**：在右侧操作区域点击"编辑"按钮
3. **修改信息**：修改广告组名称、默认出价、预算等信息
4. **保存修改**：点击"保存"按钮，系统会自动同步到亚马逊后台

### 4.3 商品广告管理

#### 4.3.1 添加商品广告

1. **选择广告组**：在左侧广告树中选择要添加商品广告的广告组
2. **点击添加商品**：在右侧操作区域点击"添加商品"按钮
3. **选择商品**：在弹出的商品选择对话框中选择要推广的商品
4. **设置出价**：为每个商品设置出价（可使用默认出价）
5. **保存添加**：点击"确定"按钮，系统会自动同步到亚马逊后台

#### 4.3.2 批量操作商品广告

1. **选择商品**：在商品广告列表中选择多个商品
2. **点击批量操作**：在列表上方点击"批量操作"按钮
3. **选择操作类型**：选择"批量修改出价"、"批量暂停"、"批量启用"等操作
4. **执行操作**：根据操作类型填写相关信息，点击"确定"按钮

### 4.4 关键词管理

#### 4.4.1 添加关键词

1. **选择广告组**：在左侧广告树中选择要添加关键词的广告组
2. **点击添加关键词**：在关键词标签页点击"添加关键词"按钮
3. **输入关键词**：在弹出的对话框中输入关键词，可批量输入多个
4. **设置匹配类型**：为每个关键词设置匹配类型（广泛、词组、精确）
5. **设置出价**：为每个关键词设置出价
6. **保存添加**：点击"确定"按钮，系统会自动同步到亚马逊后台

#### 4.4.2 优化关键词

1. **查看关键词数据**：在关键词标签页查看关键词的点击量、花费、转化率等数据
2. **分析关键词表现**：根据数据指标分析关键词的表现
3. **调整关键词**：
   - 对于表现好的关键词：提高出价，增加预算
   - 对于表现差的关键词：降低出价或暂停
   - 对于无表现的关键词：删除或调整匹配类型

### 4.5 定向管理

#### 4.5.1 添加商品定向

1. **选择广告组**：在左侧广告树中选择要添加定向的广告组
2. **点击添加定向**：在商品投放标签页点击"添加定向"按钮
3. **选择定向类型**：选择"商品定向"、"分类定向"等类型
4. **设置定向条件**：根据定向类型填写相关条件
5. **设置出价**：为定向设置出价
6. **保存添加**：点击"确定"按钮，系统会自动同步到亚马逊后台

#### 4.5.2 添加否定定向

1. **选择广告组**：在左侧广告树中选择要添加否定定向的广告组
2. **点击添加否定定向**：在广告组-否定投放标签页点击"添加否定定向"按钮
3. **输入否定条件**：输入要否定的商品ASIN或分类
4. **保存添加**：点击"确定"按钮，系统会自动同步到亚马逊后台

## 5. 数据说明

### 5.1 核心数据指标

| 指标名称 | 英文缩写 | 描述 | 数据来源 |
|---------|---------|------|----------|
| 点击量 | clicks | 广告被点击的次数 | 亚马逊广告数据 |
| 展示量 | impressions | 广告被展示的次数 | 亚马逊广告数据 |
| 点击率 | CTR | 点击量与展示量的比值 | 计算得出（点击量/展示量） |
| 花费 | cost | 广告的总花费 | 亚马逊广告数据 |
| 销售额 | sales | 广告带来的销售额 | 亚马逊广告数据 |
| 订单量 | orders | 广告带来的订单数量 | 亚马逊广告数据 |
| 转化率 | conversion | 订单量与点击量的比值 | 计算得出（订单量/点击量） |
| ACOS | ACOS | 广告花费与广告销售额的比值 | 计算得出（花费/销售额） |
| ROAS | ROAS | 广告销售额与广告花费的比值 | 计算得出（销售额/花费） |
| 平均点击花费 | CPC | 广告花费与点击量的比值 | 计算得出（花费/点击量） |
| 广告销售成本 | CAC | 广告花费与订单量的比值 | 计算得出（花费/订单量） |

### 5.2 数据更新频率

| 数据类型 | 更新频率 | 更新时间 | 延迟时间 |
|---------|---------|---------|----------|
| 点击量 | 实时 | 每小时更新 | 0-2小时 |
| 展示量 | 实时 | 每小时更新 | 0-2小时 |
| 花费 | 实时 | 每小时更新 | 0-2小时 |
| 销售额 | 延迟 | 每日更新 | 1-3天 |
| 订单量 | 延迟 | 每日更新 | 1-3天 |
| 转化率 | 延迟 | 每日更新 | 1-3天 |
| ACOS | 延迟 | 每日更新 | 1-3天 |

### 5.3 数据准确性

- 系统数据来源于亚马逊官方广告API，确保数据的真实性和准确性
- 由于亚马逊API的限制，部分数据可能存在1-3天的延迟
- 系统会定期与亚马逊后台同步数据，确保数据的一致性

## 6. 常见问题

### 6.1 广告创建失败

**问题**：创建广告活动或广告组时失败
**解决方案**：
- 检查广告名称是否符合亚马逊要求（不超过128个字符）
- 检查预算设置是否合理（最小值为1美元）
- 检查网络连接是否正常
- 检查亚马逊账户是否有足够的余额
- 联系技术支持

### 6.2 数据不同步

**问题**：系统数据与亚马逊后台数据不一致
**解决方案**：
- 等待系统自动同步（通常为1-3天）
- 手动点击"同步数据"按钮
- 检查网络连接是否正常
- 联系技术支持

### 6.3 广告状态异常

**问题**：广告状态显示异常或与亚马逊后台不一致
**解决方案**：
- 刷新页面，重新加载数据
- 手动点击"同步状态"按钮
- 检查亚马逊账户状态是否正常
- 联系技术支持

### 6.4 批量操作失败

**问题**：批量创建、编辑广告时失败
**解决方案**：
- 检查批量操作的数量是否超过限制（单次最多100个）
- 检查操作内容是否符合亚马逊要求
- 检查网络连接是否正常
- 分批进行操作
- 联系技术支持

### 6.5 权限不足

**问题**：无法执行某些广告操作
**解决方案**：
- 检查是否具备对应的广告操作权限
- 联系管理员申请权限
- 检查账户是否被禁用

## 7. 使用技巧

### 7.1 广告优化技巧

1. **关键词策略**：
   - 广泛匹配：用于发现新的关键词，扩大流量
   - 词组匹配：用于精准定位目标客户
   - 精确匹配：用于提高转化率，降低ACOS
   - 定期优化：根据数据表现，调整关键词匹配类型和出价

2. **出价策略**：
   - 新品期：设置较高的初始出价，快速获得曝光
   - 稳定期：根据转化率调整出价，优化ACOS
   - 成熟期：根据ROAS调整出价，最大化利润
   - 使用动态出价：根据竞争情况自动调整出价

3. **预算分配**：
   - 重点产品：分配更多预算，确保获得足够曝光
   - 新品推广：设置合理预算，测试效果后再调整
   - 季节性产品：根据季节变化调整预算
   - 定期检查：监控预算使用情况，避免预算过早耗尽

4. **定向优化**：
   - 商品定向：定向相关商品，提高转化率
   - 分类定向：定向相关分类，扩大流量
   - 否定定向：排除不相关的搜索词和商品，减少无效点击
   - 受众定向：针对特定受众群体，提高广告相关性

### 7.2 操作效率提升

1. **批量操作**：
   - 批量创建广告活动和广告组
   - 批量添加关键词和商品
   - 批量修改出价和预算
   - 批量暂停和启用广告

2. **模板使用**：
   - 保存常用的广告设置为模板
   - 使用模板快速创建类似的广告活动
   - 定期更新模板，适应市场变化

3. **快捷键**：
   - Ctrl+F：快速搜索广告活动或关键词
   - Ctrl+C/Ctrl+V：复制粘贴关键词和设置
   - Enter：确认操作
   - Esc：取消操作

4. **筛选功能**：
   - 使用状态筛选：快速找到暂停或启用的广告
   - 使用日期筛选：查看特定时间段的广告数据
   - 使用关键词筛选：快速找到特定关键词的广告
   - 使用表现筛选：快速找到高表现或低表现的广告

### 7.3 最佳实践

1. **广告结构优化**：
   - 每个广告活动专注于一个目标（新品推广、库存清理等）
   - 每个广告组包含相关性高的商品
   - 关键词与商品高度相关
   - 定期检查和调整广告结构

2. **数据监控**：
   - 每日：监控广告花费和基本数据
   - 每周：分析关键词表现，调整出价和预算
   - 每月：评估整体广告效果，优化广告策略
   - 季度：进行全面的广告复盘，制定长期策略

3. **A/B测试**：
   - 测试不同的广告标题和创意
   - 测试不同的关键词匹配类型
   - 测试不同的出价策略
   - 测试不同的定向方式

4. **合规运营**：
   - 遵守亚马逊广告政策
   - 避免使用禁止的关键词和创意
   - 确保广告内容真实准确
   - 定期检查广告合规性

## 9. 附录

### 9.1 广告术语解释

| 术语 | 解释 |
|------|------|
| SP广告 | Sponsored Products，针对单个商品的广告 |
| SB广告 | Sponsored Brands，品牌广告，包含品牌logo和多个商品 |
| SD广告 | Sponsored Display，展示型广告，可定向特定商品或受众 |
| ACOS | Advertising Cost of Sales，广告花费与广告销售额的比值 |
| ROAS | Return on Ad Spend，广告销售额与广告花费的比值 |
| CTR | Click Through Rate，点击率，点击量与展示量的比值 |
| CPC | Cost Per Click，每次点击成本，广告花费与点击量的比值 |
| CVR | Conversion Rate，转化率，订单量与点击量的比值 |
| CAC | Customer Acquisition Cost，客户获取成本，广告花费与订单量的比值 |
| 广泛匹配 | Broad Match，匹配与关键词相关的搜索词 |
| 词组匹配 | Phrase Match，匹配包含关键词的搜索词 |
| 精确匹配 | Exact Match，仅匹配完全一致的搜索词 |
| 动态出价 | Dynamic Bidding，根据竞争情况自动调整出价 |
| 否定关键词 | Negative Keywords，排除不相关的搜索词 |
| 商品定向 | Product Targeting，定向特定商品或分类 |
| 受众定向 | Audience Targeting，定向特定受众群体 |

### 9.2 广告政策要点

1. **禁止的内容**：
   - 违反亚马逊政策的商品
   - 虚假或误导性的广告内容
   - 侵犯知识产权的内容
   - 价格误导或促销信息

2. **广告质量要求**：
   - 广告标题清晰准确
   - 广告创意与商品相关
   - 着陆页与广告内容一致
   - 避免使用夸大的宣传语

3. **合规检查**：
   - 定期检查广告内容是否符合政策
   - 及时更新不符合政策的广告
   - 关注亚马逊政策变化
   - 遵守各国家和地区的法律法规

   - 启用双因素认证
   - 避免在公共网络上操作广告
   - 定期检查账户活动记录

2. **数据保护**：

   - 定期审查权限设置
   - 及时撤销不再需要的权限
   - 记录重要操作的审计日志
