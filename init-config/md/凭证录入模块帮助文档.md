# 凭证录入模块帮助文档

## 1. 模块概述

凭证录入模块是财务管理系统的核心功能之一，用于创建、管理和处理会计记账凭证。该模块支持完整的凭证生命周期管理，包括凭证创建、修改、审核、过账等操作，并提供实时的账簿更新和数据备份功能。

### 1.1 核心功能

- **凭证创建**：支持手动录入凭证信息和分录明细
- **凭证管理**：修改、删除、查询凭证
- **凭证审核**：审核凭证的准确性和合法性
- **凭证过账**：将凭证数据过账到总账和明细账
- **附件管理**：支持上传和管理凭证附件
- **实时更新**：凭证操作实时反映到账簿数据

### 1.2 技术特点

- **实时过账**：凭证保存时自动更新总账和明细账
- **数据备份**：操作前自动备份数据，支持异常恢复
- **事务管理**：确保凭证操作的原子性和一致性
- **借贷平衡**：自动校验凭证借贷平衡
- **会计期间控制**：防止修改已结账期间的凭证

## 2. 功能说明

### 2.1 凭证创建

**功能描述**：创建新的记账凭证，包括凭证头部信息和分录明细。

**操作步骤**：
1. 进入凭证录入页面
2. 填写凭证头部信息（凭证类型、日期、附件数量等）
3. 点击"获取凭证号"按钮生成凭证编号
4. 录入凭证分录（会计科目、借贷金额、摘要等）
5. 系统自动计算借贷总额并校验平衡
6. 上传凭证附件（可选）
7. 点击"保存"按钮提交凭证

**注意事项**：
- 凭证金额不能为空
- 借贷金额必须平衡
- 凭证日期不能早于已结账期间

### 2.2 凭证修改

**功能描述**：修改已存在的凭证信息和分录明细。

**操作步骤**：
1. 在凭证列表中找到需要修改的凭证
2. 点击"编辑"按钮进入修改页面
3. 修改凭证信息和分录明细
4. 点击"保存"按钮提交修改

**系统处理**：
- 自动备份原凭证数据
- 从账簿中移除原凭证影响
- 更新凭证数据
- 重新过账到账簿
- 记录修改日志

**限制条件**：
- 已结账期间的凭证不能修改
- 已过账的凭证需要先反过账才能修改

### 2.3 凭证删除

**功能描述**：删除指定的凭证。

**操作步骤**：
1. 在凭证列表中选择需要删除的凭证
2. 点击"删除"按钮
3. 确认删除操作

**系统处理**：
- 检查凭证状态
- 备份原凭证数据
- 从账簿中移除原凭证影响
- 删除凭证分录
- 删除凭证主数据

**限制条件**：
- 已结账期间的凭证不能删除
- 已过账的凭证需要先反过账才能删除

### 2.4 凭证审核

**功能描述**：审核凭证的准确性和合法性，将凭证状态改为已审核。

**操作步骤**：
1. 在凭证列表中选择需要审核的凭证
2. 点击"审核"按钮
3. 系统自动更新凭证状态

**注意事项**：
- 只有状态为"草稿"的凭证才能审核
- 审核后的凭证才能进行过账操作

### 2.5 凭证过账

**功能描述**：将已审核的凭证过账到总账和明细账，更新账户余额。

**操作步骤**：
1. 在凭证列表中选择需要过账的凭证
2. 点击"过账"按钮
3. 系统自动处理过账操作

**系统处理**：
- 检查凭证状态
- 校验借贷平衡
- 更新凭证状态为已过账
- 记录过账时间

**注意事项**：
- 只有已审核的凭证才能过账
- 过账后的凭证不能直接修改

### 2.6 凭证查询

**功能描述**：根据条件查询凭证列表。

**查询条件**：
- 凭证日期范围
- 凭证编号
- 凭证类型
- 凭证状态
- 制单人
- 审核人

**操作步骤**：
1. 进入凭证查询页面
2. 填写查询条件
3. 点击"查询"按钮
4. 查看查询结果

### 2.7 附件管理

**功能描述**：上传、查看和管理凭证附件。

**操作步骤**：
1. 在凭证创建或编辑页面
2. 点击"上传附件"按钮
3. 选择需要上传的文件
4. 点击"确定"按钮完成上传

**支持的文件类型**：
- 图片文件（JPG、PNG、GIF等）
- 文档文件（PDF、Word、Excel等）
- 其他文件类型

## 3. 操作指南

### 3.1 凭证录入操作流程

**1. 准备工作**
- 确认当前会计期间未结账
- 准备好需要录入的凭证信息和附件
- 确保有足够的权限进行凭证录入操作

**2. 凭证创建**
- 进入凭证录入模块
- 点击"新增"按钮
- 填写凭证头部信息
- 获取凭证编号
- 录入分录明细
- 校验借贷平衡
- 上传附件
- 保存凭证

**3. 凭证审核**
- 进入凭证列表页面
- 选择需要审核的凭证
- 点击"审核"按钮
- 确认审核操作

**4. 凭证过账**
- 进入凭证列表页面
- 选择已审核的凭证
- 点击"过账"按钮
- 确认过账操作

**5. 凭证查询**
- 进入凭证查询页面
- 设置查询条件
- 执行查询
- 查看凭证详情

### 3.2 常见操作场景

**场景一：日常费用报销**
1. 创建凭证，选择"普通凭证"类型
2. 凭证日期选择报销日期
3. 分录1：借-管理费用-办公费，金额为报销金额
4. 分录2：贷-银行存款，金额与借方一致
5. 上传报销单据作为附件
6. 保存并提交审核

**场景二：销售收款**
1. 创建凭证，选择"普通凭证"类型
2. 凭证日期选择收款日期
3. 分录1：借-银行存款，金额为收款金额
4. 分录2：贷-主营业务收入，金额为收入部分
5. 分录3：贷-应交税费-应交增值税，金额为税额部分
6. 上传银行回单作为附件
7. 保存并提交审核

**场景三：采购付款**
1. 创建凭证，选择"普通凭证"类型
2. 凭证日期选择付款日期
3. 分录1：借-原材料/库存商品，金额为采购金额
4. 分录2：借-应交税费-应交增值税（进项税额），金额为税额
5. 分录3：贷-银行存款，金额为付款总额
6. 上传采购发票和银行回单作为附件
7. 保存并提交审核

## 4. 常见问题

### 4.1 凭证金额不能为空

**问题描述**：保存凭证时提示"凭证金额不能为空"

**解决方案**：
- 检查凭证总金额是否填写
- 检查分录明细中的借贷金额是否填写
- 确保至少有一条有效的分录记录

### 4.2 凭证借贷不平衡

**问题描述**：保存凭证时提示"凭证借贷不平衡"

**解决方案**：
- 检查所有分录的借贷金额
- 确保借方总额等于贷方总额
- 检查是否有金额输入错误

### 4.3 无法修改凭证

**问题描述**：点击修改按钮时提示"凭证不能修改"

**解决方案**：
- 检查凭证是否处于已过账状态，需要先反过账
- 检查凭证所在会计期间是否已结账
- 检查是否有足够的权限修改凭证

### 4.4 无法删除凭证

**问题描述**：点击删除按钮时提示"凭证不能删除"

**解决方案**：
- 检查凭证是否处于已过账状态，需要先反过账
- 检查凭证所在会计期间是否已结账
- 检查是否有足够的权限删除凭证

### 4.5 附件上传失败

**问题描述**：上传凭证附件时失败

**解决方案**：
- 检查文件大小是否超过限制
- 检查文件格式是否支持
- 检查网络连接是否正常
- 检查服务器存储空间是否充足

## 5. 最佳实践

### 5.1 凭证录入规范

1. **凭证编号**：使用系统自动生成的凭证编号，确保编号连续且唯一
2. **凭证日期**：使用实际业务发生日期，避免跨期凭证
3. **摘要填写**：摘要应简明扼要，准确反映业务内容
4. **科目选择**：选择正确的会计科目，确保核算准确
5. **金额录入**：仔细核对金额，避免输入错误
6. **附件管理**：及时上传相关附件，确保凭证可追溯

### 5.2 凭证审核要点

1. **业务真实性**：审核业务是否真实发生
2. **凭证完整性**：检查凭证要素是否完整
3. **核算准确性**：核对科目使用和金额计算是否正确
4. **手续完备性**：检查审批手续是否齐全
5. **附件合规性**：审核附件是否合法有效

### 5.3 系统操作建议

1. **定期备份**：定期导出凭证数据作为备份
2. **及时审核**：凭证录入后及时审核，避免积压
3. **按时过账**：定期进行凭证过账，确保账簿数据及时更新
4. **期末检查**：期末结账前检查所有凭证状态
5. **权限管理**：合理设置用户权限，确保职责分离

### 5.4 故障处理

1. **系统异常**：如遇系统异常，不要重复操作，联系系统管理员
2. **数据错误**：发现数据错误时，及时通过正常流程修改，不要直接修改数据库
3. **备份恢复**：定期测试备份数据的恢复功能，确保数据安全
4. **日志查询**：遇到问题时，查看系统日志获取详细信息
