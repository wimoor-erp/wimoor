# 财务-库存模块帮助文档

## 1. 模块概述

财务-库存模块是Wimoor系统中的核心功能模块，主要负责库存数据的管理、查询和分析，为企业提供实时准确的库存信息，支持财务决策和业务运营。

### 1.1 核心功能

- **库存日报表**：查看每日库存变化情况，支持按仓库、日期、SKU等条件查询
- **基础库存报表**：查看当前库存状态，包括待入库、可用库存、待出库等信息
- **库存周转分析**：分析库存周转情况，优化库存管理
- **滞销库存分析**：识别滞销产品，减少库存积压

### 1.2 应用场景

- **日常库存监控**：实时了解库存状态，及时发现库存异常
- **财务核算**：为成本核算、资产负债表编制提供准确的库存数据
- **采购决策**：根据库存水平和周转情况，制定合理的采购计划
- **销售策略**：基于库存状况，调整销售策略和促销活动

## 2. 功能详细说明

### 2.1 库存日报表

#### 2.1.1 功能介绍

库存日报表用于查看指定时间段内的库存每日变化情况，支持按仓库、日期和SKU进行筛选，并提供数据导出功能。

#### 2.1.2 操作步骤

1. **进入模块**：点击左侧菜单「仓库管理」→「库存管理」→「库存日报表」

2. **选择仓库**：在仓库选择器中选择要查看的仓库

3. **选择日期**：点击日期选择器，选择要查询的日期范围（默认为1天）

4. **输入SKU**：在搜索框中输入SKU，点击搜索按钮进行精确查询

5. **查看数据**：系统会显示符合条件的库存日报表数据，包括产品信息、仓库、每日库存变化等

6. **导出数据**：点击「导出」按钮，系统会生成Excel格式的库存日报表

#### 2.1.3 注意事项

- 日期范围默认限制为1天，如需查看更长时间范围，请手动调整
- 导出功能支持大数量数据导出，系统会自动处理

### 2.2 基础库存报表

#### 2.2.1 功能介绍

基础库存报表用于查看当前库存状态，包括待入库、可用库存、待出库等详细信息，支持按供应商、仓库、产品分类等条件筛选。

#### 2.2.2 操作步骤

1. **进入模块**：点击左侧菜单「仓库管理」→「库存管理」→「基础库存报表」

2. **选择供应商**：在供应商选择器中选择特定供应商（可选）

3. **选择仓库**：在仓库选择器中选择要查看的仓库

4. **输入SKU**：在搜索框中输入SKU，点击搜索按钮进行查询

5. **高级筛选**：点击筛选按钮，可按子产品SKU、辅料SKU、产品分类、产品类型等进行高级筛选

6. **查看数据**：系统会显示符合条件的库存数据，包括产品信息、仓库、待入库、可用库存、待出库等

7. **查看详情**：鼠标悬停在「待入库」或「待出库」数值上，可查看详细的入库或出库信息

8. **修复库存**：在详情弹窗中，点击刷新图标可修复库存数据（此操作存在风险，谨慎使用）

9. **导出数据**：点击「导出」按钮，系统会生成Excel格式的库存报表

#### 2.2.3 注意事项

- 「精确查询」选项可用于精确匹配SKU
- 「仅查看有库存产品」选项可用于过滤掉无库存的产品
- 修复库存功能仅在确定库存数据错误时使用

### 2.3 库存周转分析

#### 2.3.1 功能介绍

库存周转分析用于分析库存的周转情况，帮助企业优化库存管理，减少库存积压，提高资金利用效率。

#### 2.3.2 操作步骤

1. **进入模块**：点击左侧菜单「仓库管理」→「库存管理」→「库存周转分析」

2. **选择仓库**：在仓库选择器中选择要分析的仓库

3. **选择时间范围**：选择要分析的时间范围

4. **查看分析结果**：系统会显示库存周转率、周转天数等关键指标

### 2.4 滞销库存分析

#### 2.4.1 功能介绍

滞销库存分析用于识别长期未销售的产品，帮助企业及时处理滞销库存，减少资金占用。

#### 2.4.2 操作步骤

1. **进入模块**：点击左侧菜单「仓库管理」→「库存管理」→「滞销库存分析」

2. **选择仓库**：在仓库选择器中选择要分析的仓库

3. **设置滞销天数**：设置判断滞销的天数阈值

4. **查看分析结果**：系统会显示符合条件的滞销产品列表

## 3. 常见问题与解决方案

### 3.1 库存数据不准确

**问题**：系统显示的库存数据与实际库存不符

**解决方案**：
1. 检查是否有未处理的入库或出库单据
2. 使用「修复库存」功能修复库存数据
3. 联系系统管理员进行数据同步

### 3.2 导出数据失败

**问题**：点击导出按钮后，系统没有生成Excel文件

**解决方案**：
1. 检查网络连接是否正常
2. 检查浏览器是否阻止了弹出窗口
3. 尝试缩小导出数据范围，分批导出

### 3.3 无法查询到特定产品的库存

**问题**：在库存报表中无法查询到特定产品的库存信息

**解决方案**：
1. 检查产品是否已添加到系统中
2. 检查产品是否分配到了正确的仓库
3. 检查查询条件是否设置正确

### 3.4 库存周转分析数据异常

**问题**：库存周转率计算结果异常

**解决方案**：
1. 检查是否有足够的历史销售数据
2. 检查产品的成本数据是否正确
3. 联系系统管理员进行数据检查

## 4. 最佳实践

### 4.1 库存管理建议

1. **定期盘点**：定期进行库存盘点，确保系统数据与实际库存一致
2. **设置安全库存**：为重要产品设置安全库存，避免缺货
3. **优化库存结构**：根据销售情况，调整库存结构，减少滞销库存
4. **合理采购**：基于库存周转情况，制定合理的采购计划

### 4.2 数据利用建议

1. **结合销售数据**：将库存数据与销售数据结合分析，优化库存水平
2. **财务分析**：利用库存数据进行成本核算和财务分析
3. **供应链优化**：基于库存周转情况，优化供应链管理
4. **报表分析**：定期生成库存分析报表，为管理决策提供依据
