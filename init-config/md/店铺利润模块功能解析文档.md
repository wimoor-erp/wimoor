# 店铺利润模块功能解析文档

## 一、模块概述

店铺利润模块是 Wimoor ERP 系统中用于统计和分析亚马逊店铺利润的核心模块。该模块按店铺维度统计销售数据、费用支出和最终利润，支持多币种转换和数据导出功能。

### 1.1 功能定位

| 功能维度 | 说明 |
| :--- | :--- |
| **核心目标** | 按店铺统计销售利润，帮助商家了解各店铺经营状况 |
| **数据来源** | 亚马逊结算报告（Settlement Report） |
| **统计维度** | 按店铺/站点分组统计 |
| **币种支持** | 支持人民币(CNY)、美元(USD)、站点币种(market) |

```javascript
    }
```

```java
    Map<String, Object> map = new HashMap<>();
    map.put("fromDate", fromDate);
    map.put("endDate", toDate);
    map.put("groupid", groupid);
    map.put("marketplaceid", marketplaceid);

    return Result.success(list);
}
```

#### 3.2.2 利润计算公式

| 计算项 | 公式 | 说明 |
| :--- | :--- | :--- |
| **结算收入** | `setin` | 来自亚马逊结算报告 |

| **其他费用** | `setin - storagefee - advfee - refund - fbafee - commission - principal` | 差额计算 |

## 四、数据流程图

```
┌─────────────────────┐     ┌─────────────────────┐     ┌─────────────────────┐
│   前端请求          │────▶│   Controller        │────▶│   Service           │
│  (queryParams)      │     │  (参数校验/组装)     │     │  (业务逻辑处理)     │
└─────────────────────┘     └─────────────────────┘     └─────────────────────┘
                                                              │
                                                              ▼
┌─────────────────────┐     ┌─────────────────────┐     ┌─────────────────────┐
│   前端展示          │◀────│   Service           │◀────│   Mapper            │
│  (tableData)        │     │  (数据汇总/转换)     │     │  (数据库查询)       │
└─────────────────────┘     └─────────────────────┘     └─────────────────────┘
```

### 4.1 数据处理流程

1. **参数接收**：前端传入查询条件（日期范围、店铺、币种等）
2. **参数校验**：后端校验并补全默认值（如默认近7天）
3. **数据查询**：从 `amz_settlement_summary_sku` 表查询结算数据
4. **货币转换**：根据目标币种进行汇率转换
5. **成本计算**：查询采购成本和其他费用
6. **利润计算**：结算收入 - 采购成本
7. **分组汇总**：按店铺分组并计算合计
8. **结果返回**：返回格式化后的利润数据

## 五、关键技术点

### 5.1 多币种支持

系统支持三种币种显示：
- **CNY**：人民币
- **USD**：美元
- **market**：站点原币种

转换逻辑：
```java
if("market".equals(currency)) {
    currency = market.getCurrency();  // 使用站点默认币种
```

## 七、总结

店铺利润模块是一个数据统计分析模块，核心功能包括：

1. **多维度筛选**：支持按店铺、站点、日期范围筛选
2. **多币种展示**：支持人民币、美元、站点币种切换
3. **利润计算**：自动计算销售额、费用、成本和利润
4. **数据导出**：支持Excel格式导出
5. **权限控制**：支持按用户权限限制数据访问

该模块的数据来源于亚马逊结算报告，通过定时任务同步数据，确保利润统计的准确性和时效性。
