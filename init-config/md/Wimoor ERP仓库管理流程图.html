详细流程说明

1. 仓库创建与管理

功能说明：创建和管理仓库信息，设置仓库属性和类型。

操作步骤：

  1. 进入仓库管理页面（路径：ERP → 仓库 → 仓库管理）
  2. 点击"创建仓库"按钮
  3. 填写仓库名称、代码、地址等基本信息
  4. 选择仓库类型（本地仓、FBA仓等）
  5. 设置仓库属性（是否默认仓库、是否启用等）
  6. 提交保存
  7. 仓库创建成功后，可在仓库列表中查看和管理

2. 库位管理

功能说明：在仓库中创建和管理库位，用于更精细的库存管理。

操作步骤：

  1. 进入库位管理页面（路径：ERP → 仓库 → 库位管理）
  2. 选择要管理的仓库
  3. 点击"创建库位"按钮
  4. 填写库位编码、名称、位置等信息
  5. 设置库位属性（是否启用、是否允许负库存等）
  6. 提交保存
  7. 库位创建成功后，可在库位列表中查看和管理

3. 库存入库流程

功能说明：处理货物入库，增加库存数量。

操作步骤：

  1. 进入入库管理页面（路径：ERP → 仓库 → 入库管理）
  2. 选择入库类型（采购入库、生产入库、其他入库）
  3. 填写入库单基本信息（单号、日期、供应商等）
  4. 添加入库商品及数量
  5. 选择入库库位
  6. 确认入库信息
  7. 提交保存，系统自动更新库存

4. 库存出库流程

功能说明：处理货物出库，减少库存数量。

操作步骤：

  1. 进入出库管理页面（路径：ERP → 仓库 → 出库管理）
  2. 选择出库类型（销售出库、生产出库、其他出库）
  3. 填写出库单基本信息（单号、日期、客户等）
  4. 添加出库商品及数量
  5. 选择出库库位
  6. 确认出库信息
  7. 提交保存，系统自动更新库存

5. 库存调整

功能说明：对库存数量进行调整，处理库存差异。

操作步骤：

  1. 进入库存调整页面（路径：ERP → 仓库 → 库存调整）
  2. 选择调整类型（盘盈、盘亏）
  3. 选择仓库和库位
  4. 添加需要调整的商品及数量
  5. 填写调整原因
  6. 提交保存，系统自动更新库存

注意事项

  1. 库存影响：入库会增加库存，出库会减少库存，调整会根据类型增减库存
  2. 库位管理：库位是仓库的子单位，用于更精细的库存管理
  3. 数据同步：库存数据会与其他模块（如采购、销售）同步
  4. 权限控制：不同角色的用户有不同的仓库操作权限
  5. 审核流程：部分入库和出库操作可能需要审核
