# 财务-库存货值模块帮助文档

## 1. 模块概述

财务-库存货值模块是Wimoor系统中的重要功能模块，主要负责计算和展示企业的库存货值，包括本地仓和FBA仓的库存价值，为企业提供准确的库存资产信息，支持财务决策和业务运营。

### 1.1 核心功能

- **本地仓库存货值**：计算和展示本地仓库的库存价值，包括现有库存、待入库库存、在途货值等
- **FBA仓库存货值**：计算和展示FBA仓库的库存价值，包括可用库存、预留库存、待入库库存等
- **数据筛选**：支持按日期、仓库、SKU等条件筛选库存货值数据
- **数据导出**：支持导出库存货值数据为Excel文件
- **合计功能**：自动计算库存货值的合计金额

### 1.2 应用场景

- **财务核算**：为资产负债表编制提供准确的库存资产数据
- **库存管理**：了解库存价值分布，优化库存结构
- **成本控制**：监控库存成本，控制采购支出
- **决策支持**：基于库存货值数据，制定合理的采购和销售策略

## 2. 功能详细说明

### 2.1 本地仓库存货值

#### 2.1.1 功能介绍

本地仓库存货值用于查看本地仓库的库存价值情况，包括现有库存、待入库库存、采购成本单价、总货值、在途货值、采购待付款和其它成本等信息。

#### 2.1.2 操作步骤

1. **进入模块**：点击左侧菜单「财务」→「库存货值」→「本地仓」

2. **选择仓库**：在仓库选择器中选择要查看的仓库

3. **选择日期**：点击日期选择器，选择要查询的库存日期（默认为当天）

4. **输入SKU**：在搜索框中输入本地SKU，点击搜索按钮进行查询

5. **查看数据**：系统会显示符合条件的库存货值数据，包括产品信息、现有库存、待入库库存、采购成本单价、总货值、在途货值、采购待付款和其它成本等

6. **导出数据**：点击「导出」按钮，系统会生成Excel格式的库存货值报表

#### 2.1.3 注意事项

- 当选择当天日期时，系统会显示待入库库存、在途货值和采购待付款信息
- 当选择历史日期时，系统只会显示现有库存、采购成本单价、总货值和其它成本信息

### 2.2 FBA仓库存货值

#### 2.2.1 功能介绍

FBA仓库存货值用于查看FBA仓库的库存价值情况，包括可用库存、预留库存、待入库库存、总库存、采购单价、总货值、其它费用和头程运费等信息。

#### 2.2.2 操作步骤

1. **进入模块**：点击左侧菜单「财务」→「库存货值」→「FBA仓」

2. **选择店铺**：在店铺选择器中选择要查看的店铺

3. **选择日期**：点击日期选择器，选择要查询的库存日期（默认为当天）

4. **输入SKU**：在搜索框中输入平台SKU，点击搜索按钮进行查询

5. **查看数据**：系统会显示符合条件的FBA库存货值数据，包括产品信息、可用库存、预留库存、待入库库存、总库存、采购单价、总货值、其它费用和头程运费等

6. **导出数据**：点击「导出」按钮，系统会生成Excel格式的FBA库存货值报表

#### 2.2.3 注意事项

- FBA仓库存货值数据来源于Amazon API同步的数据
- 头程运费会分摊到每个产品的成本中

## 3. 数据说明

### 3.1 本地仓库存货值字段说明

| 字段名称 | 说明 | 计算方式 |
|---------|------|----------|
| 现有库存 | 当前仓库中的实际库存数量 | 可用库存 + 待出库库存 |
| 待入库库存 | 已经采购但尚未入库的库存数量 | 采购订单中未入库的数量 |
| 采购成本单价 | 产品的采购成本单价 | 采购总成本 / 采购数量 |
| 总货值 | 现有库存的总价值 | 现有库存 × 采购成本单价 |
| 在途货值 | 已经采购但尚未入库的货物价值 | 待入库库存 × 采购成本单价 |
| 采购待付款 | 尚未支付的采购款项 | 待入库库存 × 采购成本单价 |
| 其它成本 | 除采购成本外的其它成本 | 包括运输费、税费等 |

### 3.2 FBA仓库存货值字段说明

| 字段名称 | 说明 | 计算方式 |
|---------|------|----------|
| 可用库存 | FBA仓库中可销售的库存数量 | Amazon API提供的数据 |
| 预留库存 | FBA仓库中已被订单占用的库存数量 | Amazon API提供的数据 |
| 待入库库存 | 正在运往FBA仓库的库存数量 | Amazon API提供的数据 |
| 总库存 | FBA仓库中的总库存数量 | 可用库存 + 预留库存 |
| 采购单价 | 产品的采购成本单价 | 采购总成本 / 采购数量 |
| 总货值 | FBA库存的总价值 | 总库存 × 采购成本单价 |
| 其它费用 | 除采购成本外的其它费用 | 包括处理费、仓储费等 |
| 头程运费 | 产品从国内运往FBA仓库的运费 | 总头程运费分摊到每个产品 |

## 4. 常见问题与解决方案

### 4.1 库存货值数据不准确

**问题**：系统显示的库存货值与实际情况不符

**解决方案**：
1. 检查采购成本数据是否正确录入
2. 检查库存数量是否与实际库存一致
3. 检查头程运费分摊是否准确
4. 联系系统管理员进行数据同步

### 4.2 导出数据失败

**问题**：点击导出按钮后，系统没有生成Excel文件

**解决方案**：
1. 检查网络连接是否正常
2. 检查浏览器是否阻止了弹出窗口
3. 尝试缩小导出数据范围，分批导出

### 4.3 无法查询到特定产品的库存货值

**问题**：在库存货值报表中无法查询到特定产品的信息

**解决方案**：
1. 检查产品是否已添加到系统中
2. 检查产品是否分配到了正确的仓库
3. 检查采购成本数据是否已录入
4. 检查查询条件是否设置正确

### 4.4 FBA库存数据不同步

**问题**：FBA库存货值数据与Amazon后台显示的数据不一致

**解决方案**：
1. 手动触发数据同步
2. 检查Amazon API授权是否有效
3. 等待系统自动同步（通常每小时同步一次）
4. 联系系统管理员进行手动同步
