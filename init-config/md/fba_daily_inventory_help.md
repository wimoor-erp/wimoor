# FBA每日库存模块帮助文档

## 1. 模块概述

FBA每日库存模块是Wimoor系统中用于展示和分析亚马逊FBA仓库每日库存变化的核心功能模块。该模块提供了直观的表格展示，支持多维度筛选和数据导出，帮助卖家实时掌握库存动态，优化库存管理策略。

## 2. 功能特点

### 2.1 核心功能

- **动态日期列展示**：根据选择的日期范围，自动生成每日库存数据列
- **多维度筛选**：支持按产品组、日期范围、SKU等条件筛选数据
- **实时库存数据**：展示每个SKU在不同仓库的每日库存变化
- **数据导出功能**：支持将库存数据导出为Excel文件
- **合计行统计**：自动计算每列的库存合计值
- **产品信息展示**：包含产品图片、名称和SKU等详细信息

### 2.2 技术优势

- **高性能数据处理**：采用虚拟滚动技术，支持处理大量库存数据
- **灵活的日期范围选择**：支持自定义日期范围，默认显示最近7天数据
- **响应式设计**：适配不同屏幕尺寸，提供良好的移动端体验
- **实时数据更新**：数据自动刷新，确保信息时效性

## 3. 使用方法

### 3.1 模块入口

1. 登录Wimoor系统
2. 进入亚马逊模块
3. 选择"库存管理"菜单
4. 点击"FBA每日库存"进入模块

### 3.2 界面说明

#### 3.2.1 顶部操作栏

- **产品组选择器**：选择要查看的产品组
- **日期选择器**：设置要查询的日期范围
- **SKU搜索框**：输入SKU关键词进行搜索
- **导出按钮**：将当前数据导出为Excel文件

#### 3.2.2 数据表格

- **产品信息列**：显示产品图片、名称和SKU
- **仓库列**：显示产品所在的FBA仓库
- **日期列**：根据选择的日期范围自动生成，显示每日库存数量
- **合计行**：位于表格底部，显示每列的合计值

## 4. 操作步骤

### 4.1 基础查询操作

1. **选择产品组**：点击产品组选择器，从下拉列表中选择要查看的产品组
2. **设置日期范围**：点击日期选择器，选择起始日期和结束日期
3. **搜索SKU**：在搜索框中输入SKU关键词，点击搜索按钮或按回车键
4. **查看数据**：系统自动加载并展示符合条件的库存数据

### 4.2 高级筛选操作

1. **组合筛选**：同时使用产品组、日期范围和SKU进行组合筛选
2. **排序操作**：点击列标题可以对数据进行升序或降序排序
3. **清除筛选**：点击搜索框右侧的清除按钮，清除当前筛选条件

### 4.3 数据导出操作

1. **设置导出条件**：根据需要设置筛选条件，确保显示的数据符合导出要求
2. **点击导出按钮**：点击顶部操作栏中的"导出"按钮
3. **保存文件**：系统自动生成Excel文件，选择保存路径即可完成导出

## 5. 功能详解

### 5.1 动态日期列生成

系统根据选择的日期范围，自动生成对应日期的库存列。例如，如果选择了7天的日期范围，表格将显示7个日期列，每个列对应一天的库存数据。

### 5.2 库存数据计算

每日库存数据来源于亚马逊API同步的FBA库存报告，系统自动计算并存储每日库存快照，确保数据准确性和完整性。

### 5.3 合计行功能

合计行位于表格底部，自动计算每列的库存合计值，帮助卖家快速了解整体库存情况。合计行仅在第一页数据中显示。

## 6. 常见问题

### 6.1 为什么没有数据显示？

- 检查是否选择了正确的产品组
- 检查日期范围是否正确，建议先尝试查看最近7天的数据
- 确认该产品组下是否有FBA库存数据

### 6.2 导出的数据不完整怎么办？

- 检查筛选条件是否设置正确
- 尝试缩小日期范围，减少单次导出的数据量
- 确认网络连接是否稳定

### 6.3 日期列显示异常怎么办？

- 尝试重新选择日期范围
- 刷新页面后重新查询
- 联系系统管理员检查数据同步情况

### 6.4 为什么有些SKU没有显示图片？

- 检查产品信息是否完整
- 确认产品图片是否已上传到系统
- 联系运营人员补充产品图片信息

## 7. 最佳实践

1. **定期查看库存趋势**：建议每周查看一次库存趋势，及时发现异常情况
2. **结合销售数据分析**：将每日库存数据与销售数据结合分析，优化库存周转
3. **设置合理的库存预警**：根据历史库存数据，设置合理的库存预警值
4. **定期导出数据备份**：重要的库存数据建议定期导出备份，防止数据丢失
