# 财务-采购记账模块（采购收支统计）帮助文档

## 1. 模块概述

财务-采购记账模块（采购收支统计）是Wimoor系统中的重要功能模块，主要负责统计采购支出和收入的流水账目，并通过图表展示年度和月度的收支统计，为企业提供全面的采购财务分析能力。

### 1.1 核心功能

- **采购收支统计**：统计采购支出和收入的流水账目
- **多维度汇总**：支持按SKU和供应商维度汇总采购数据
- **收支明细**：展示采购金额、付款金额、待付款金额等详细信息
- **入库信息**：展示入库数量、待入库数、退货数等入库相关信息
- **图表展示**：通过图表展示年度和月度的收支统计
- **数据导出**：支持导出采购收支报表

### 1.2 应用场景

- **采购成本分析**：分析采购成本构成，优化采购策略
- **供应商管理**：评估供应商的价格和服务质量
- **预算控制**：监控采购预算的执行情况
- **财务报表**：为财务报表提供采购数据支持
- **决策支持**：基于采购数据制定业务决策

## 2. 功能详细说明

### 2.1 采购收支统计

#### 2.1.1 功能介绍

采购收支统计功能用于统计采购支出和收入的流水账目，包括采购金额、付款金额、待付款金额等详细信息。

#### 2.1.2 操作步骤

1. **进入模块**：点击左侧菜单「采购」→「采购报表」→「采购统计」

2. **选择汇总维度**：
   - 点击「按SKU汇总」标签，按产品SKU维度查看采购数据
   - 点击「按供应商汇总」标签，按供应商维度查看采购数据

3. **设置筛选条件**：
   - 选择仓库：点击仓库选择器，选择要查看的仓库
   - 选择日期范围：点击日期选择器，选择要查询的日期范围（默认为最近30天）
   - 输入搜索关键词：在搜索框中输入SKU或供应商名称，点击搜索按钮

4. **查看数据**：系统会显示符合条件的采购收支数据，包括采购数量、采购金额、付款金额、待付款金额等详细信息

5. **导出数据**：点击「导出」按钮，系统会生成Excel格式的采购收支报表

### 2.2 图表展示

#### 2.2.1 功能介绍

图表展示功能用于通过图表形式展示年度和月度的收支统计，直观呈现采购收支趋势。

#### 2.2.2 操作步骤

1. **进入模块**：点击左侧菜单「采购」→「采购报表」→「采购统计」

2. **选择时间范围**：
   - 选择年度：在日期选择器中选择完整的年度
   - 选择月度：在日期选择器中选择特定月份

3. **查看图表**：系统会生成相应的收支统计图表，包括：
   - 年度收支趋势图：展示全年各月份的采购收支情况
   - 月度收支明细图：展示当月的采购收支明细
   - 供应商收支分布图：展示不同供应商的采购支出分布
   - SKU收支分布图：展示不同SKU的采购支出分布

4. **分析数据**：通过图表分析采购收支趋势，识别异常情况，优化采购策略

### 2.3 收支明细查询

#### 2.3.1 功能介绍

收支明细查询功能用于查看采购收支的详细记录，包括采购订单、付款记录、入库记录等。

#### 2.3.2 操作步骤

1. **进入模块**：点击左侧菜单「采购」→「采购报表」→「采购统计」

2. **设置筛选条件**：
   - 选择仓库：点击仓库选择器，选择要查看的仓库
   - 选择日期范围：点击日期选择器，选择要查询的日期范围
   - 输入搜索关键词：在搜索框中输入SKU或供应商名称

3. **查看明细**：系统会显示符合条件的采购收支明细，包括：
   - 采购订单信息：订单号、采购日期、供应商
   - 付款信息：付款日期、付款金额、付款方式
   - 入库信息：入库日期、入库数量、入库状态

4. **导出明细**：点击「导出」按钮，系统会生成Excel格式的采购收支明细报表

## 3. 数据说明

### 3.1 采购收支指标

| 指标名称 | 说明 | 计算方法 |
|---------|------|----------|
| 采购数量 | 已审核的采购订单上的产品采购数量 | 实际入库数量 + 入库差额 + 待入库数 |
| 采购金额 | 已审核的采购订单上的产品采购金额 | 采购数量 × 采购单价 |
| 采购单价 | 已审核的采购订单上产品的单价平均值 | 采购金额 ÷ 采购数量 |
| 付款金额 | 已审核的采购订单中已经付款的采购金额 | 实际付款金额 |
| 待付款金额 | 待付款金额 = 待申请付款 + 已申请付款 | 采购金额 - 付款金额 |
| 实际单价 | 已审核且入库、付款都执行完成的采购订单里，按实际支出的采购金额与实际入库的产品数量计算 | 实际付款金额 ÷ 实际入库数量 |
| 采购运费 | 已审核的采购订单的运费总额 | 订单中的运费金额 |
| 其它费用 | 款项退收时所填写的所有自定义费用项目的总和 | 各项费用之和 |
| 付款差额 | 已审核且付款已完成的采购订单中，实际付款的金额与其订单上原金额的差额 | 实际付款金额 - 订单原金额 |

### 3.2 入库相关指标

| 指标名称 | 说明 | 计算方法 |
|---------|------|----------|
| 入库(不含退货) | 已审核的采购订单中已经入库的产品数量(不含退货) | 实际入库数量 |
| 入库(含退货) | 已审核的采购订单中所有的入库数量 | 实际入库数量 + 退货数量 |
| 待入库数 | 已审核但入库未完成的采购订单中待入库的产品数量 | 采购数量 - 实际入库数量 |
| 退货数 | 已审核的采购订单的退货总数 | 退货单中记录的数量 |
| 入库差额 | 已审核且入库已完成的采购订单中，实际入库的产品数量与其订单上原采购数量的差额 | 实际入库数量 - 订单原采购数量 |

## 4. 常见问题与解决方案

### 4.1 数据统计不准确

**问题**：采购收支统计数据与实际情况不符

**解决方案**：
1. 检查筛选条件是否设置正确
2. 检查采购订单是否已审核
3. 检查付款记录是否已录入
4. 检查入库记录是否已完成
5. 联系系统管理员进行数据同步

### 4.2 图表显示异常

**问题**：收支统计图表显示异常或数据缺失

**解决方案**：
1. 检查日期范围是否设置正确
2. 检查是否有足够的数据生成图表
3. 刷新页面重新加载数据
4. 联系系统管理员检查图表配置

### 4.3 导出报表失败

**问题**：点击导出按钮后，系统没有生成Excel文件

**解决方案**：
1. 检查网络连接是否正常
2. 检查浏览器是否阻止了弹出窗口
3. 尝试缩小导出数据范围，分批导出
4. 联系系统管理员检查导出功能

### 4.4 无法查询到特定供应商的数据

**问题**：在采购统计中无法查询到特定供应商的采购数据

**解决方案**：
1. 检查供应商名称是否输入正确
2. 检查日期范围是否包含该供应商的采购记录
3. 检查供应商是否已添加到系统中
4. 联系系统管理员检查数据是否存在
