# 发货详情模块帮助文档

## 1. 模块概述

发货详情模块是Wimoor系统中用于查看和管理亚马逊FBA货件详细信息的核心功能模块。该模块以弹窗形式展示，包含货件详情、物流出库、海关申报和接收详情四个标签页，提供了全面的货件信息查询和操作功能，帮助卖家高效管理FBA发货流程。

## 2. 功能特点

### 2.1 核心功能

- **多标签页展示**：分为货件详情、物流出库、海关申报和接收详情四个标签页
- **全面的货件信息**：展示单据信息、运输信息、地址信息等完整货件数据
- **灵活的标签打印**：支持下载和打印多种类型的箱子标签
- **配货单打印**：支持简洁版和详细版配货单打印
- **装箱详情管理**：支持查看和下载装箱详情
- **到货时间确认**：支持选择和确认预计到货时间
- **实时状态更新**：展示货件的最新状态信息

### 2.2 技术优势

- **响应式设计**：适配不同屏幕尺寸，提供良好的移动端体验
- **实时数据同步**：数据自动刷新，确保信息时效性
- **高效的操作流程**：简洁明了的操作界面，减少操作步骤
- **丰富的导出选项**：支持多种格式的文件导出

## 3. 使用方法

### 3.1 模块入口

1. 登录Wimoor系统
2. 进入ERP模块
3. 选择"发货管理"菜单
4. 在发货列表中找到需要查看的货件
5. 点击"详情"按钮进入发货详情模块

### 3.2 界面说明

#### 3.2.1 顶部标签页

- **货件详情**：展示货件的基本信息、运输信息和地址信息
- **物流出库**：管理物流出库相关操作
- **海关申报**：处理海关申报相关事务
- **接收详情**：查看货件接收情况

#### 3.2.2 货件详情标签页

##### 单据信息卡片

- **订单编码**：显示发货计划的订单编码
- **货件名称**：显示FBA货件的名称
- **货件编码**：显示亚马逊分配的货件确认ID
- **货件状态**：显示货件的当前状态
- **店铺**：显示货件所属的店铺信息
- **收货站点**：显示货件的收货国家/地区
- **发货仓库**：显示货件的发货仓库
- **备注**：显示货件的备注信息

##### 运输信息卡片

- **总货值**：显示货件的总价值
- **物流总费用**：显示物流运输的总费用
- **实际发货总数**：显示实际发货的总数量
- **SKU个数**：显示货件包含的SKU数量
- **实际结算重量**：显示实际结算的重量
- **产品净重**：显示产品的净重
- **箱子重**：显示箱子的重量
- **体积**：显示货件的总体积
- **亚马逊运输方式**：显示选择的亚马逊运输方式
- **装箱箱数**：显示装箱的总箱数

##### 地址信息卡片

- **发货地址**：显示货件的发货地址
- **收货地址**：显示货件的收货地址
- **箱标收货地址**：显示箱标上的收货地址（可添加）

### 3.3 操作指南

#### 3.3.1 查看货件详情

1. 进入发货详情模块
2. 在"货件详情"标签页查看货件的基本信息
3. 点击不同卡片查看详细信息

#### 3.3.2 打印箱子标签

1. 进入"货件详情"标签页
2. 在"打印箱子标签"区域选择标签类型
3. 点击"打印箱子标签"按钮，选择需要的操作：
   - 下载箱子标签：下载标准箱子标签
   - 下载2D条形码：下载2D格式的条形码
   - 下载托盘标签：下载托盘标签

#### 3.3.3 打印配货单

1. 进入"货件详情"标签页
2. 在"打印配货单"区域选择：
   - 直接点击按钮打印简洁版配货单
   - 点击下拉箭头选择打印详细版配货单

#### 3.3.4 下载装箱详情

1. 进入"货件详情"标签页
2. 点击"下载装箱详情"按钮
3. 保存生成的装箱详情文件

#### 3.3.5 确认到货时间

1. 进入"货件详情"标签页
2. 在"预计到货时间"下拉菜单中选择合适的到货时间窗口
3. 系统自动保存选择的到货时间

#### 3.3.6 添加箱标地址

1. 进入"货件详情"标签页
2. 在"地址信息"卡片点击"添加箱标地址"按钮
3. 在弹出的对话框中填写箱标收货地址
4. 点击"确定"保存箱标地址

## 4. 各标签页功能详解

### 4.1 货件详情

- **核心功能**：查看货件的基本信息、运输信息和地址信息
- **主要操作**：打印标签、下载装箱详情、确认到货时间
- **适用场景**：货件创建后，需要查看货件基本信息和进行基本操作

### 4.2 物流出库

- **核心功能**：管理物流出库相关操作
- **主要操作**：生成物流信息、打印物流单据、确认出库
- **适用场景**：货件准备出库时，需要处理物流相关事务

### 4.3 海关申报

- **核心功能**：处理海关申报相关事务
- **主要操作**：填写申报信息、上传申报文件、跟踪申报状态
- **适用场景**：跨境发货时，需要进行海关申报

### 4.4 接收详情

- **核心功能**：查看货件接收情况
- **主要操作**：查看接收状态、处理接收异常、确认接收完成
- **适用场景**：货件发出后，需要跟踪接收情况

## 5. 常见问题

### 5.1 为什么无法打印箱子标签？

- 检查货件状态是否允许打印标签
- 确认货件已经通过审核
- 检查网络连接是否正常
- 确认浏览器是否支持PDF文件下载

### 5.2 如何修改货件的预计到货时间？

- 进入"货件详情"标签页
- 在"预计到货时间"下拉菜单中重新选择合适的到货时间窗口
- 系统会自动保存新的到货时间

### 5.3 如何添加或修改箱标收货地址？

- 进入"货件详情"标签页
- 在"地址信息"卡片点击"添加箱标地址"按钮
- 在弹出的对话框中填写新的箱标收货地址
- 点击"确定"保存修改

### 5.4 如何下载装箱详情？

- 进入"货件详情"标签页
- 点击"下载装箱详情"按钮
- 选择保存路径，即可下载装箱详情文件

### 5.5 为什么某些操作按钮是灰色的？

- 操作按钮可能受到货件状态的限制
- 检查货件是否处于可操作状态
- 确认当前用户是否有操作权限

## 6. 最佳实践

1. **及时确认到货时间**：在创建货件后，尽快确认预计到货时间，以便亚马逊安排仓库接收
2. **提前打印标签**：在货件准备出库前，提前打印好箱子标签和配货单
3. **定期查看货件状态**：定期查看货件的最新状态，及时处理异常情况
4. **保存相关文件**：下载并保存装箱详情、配货单等重要文件，以备后续查询
5. **准确填写箱标地址**：确保箱标地址准确无误，避免影响货件接收
