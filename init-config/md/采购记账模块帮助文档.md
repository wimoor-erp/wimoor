# 采购记账模块帮助文档

## 1. 模块概述

采购记账模块是 Wimoor ERP 系统中用于管理采购账户资产及收支和结余情况的功能模块。该模块提供了账户管理、收支记录、余额查询和数据可视化等功能，帮助用户实时了解采购账户的财务状况。

## 2. 功能说明

### 2.1 核心功能

- **账户管理**：创建、编辑、删除和恢复采购账户
- **余额管理**：实时展示账户资产余额
- **收支记录**：记录和查询账户的收支情况
- **数据可视化**：通过图表展示账户收支趋势和分布
- **支付方式管理**：管理不同的支付方式和默认账户设置

### 2.2 界面元素

| 元素名称 | 功能描述 | 操作说明 |
|---------|---------|--------|
| 账户列表 | 展示所有采购账户及其余额 | 点击账户卡片查看详细信息 |
| 总账户卡片 | 展示所有账户的总余额 | 点击查看整体收支情况 |
| 添加账户按钮 | 创建新的采购账户 | 点击弹出账户创建对话框 |
| 账户操作菜单 | 对账户进行重命名、删除、设置默认等操作 | 点击账户卡片右上角的更多按钮 |
| 查看删除列表按钮 | 查看已删除的账户 | 点击后弹出已删除账户列表 |
| 收支趋势图 | 展示账户收支的时间趋势 | 实时更新数据 |
| 收支分布图 | 展示不同类型收支的分布情况 | 实时更新数据 |
| 收支明细表 | 展示账户的详细收支记录 | 支持分页和筛选 |

### 2.3 表格字段说明

| 字段名称 | 说明 | 数据来源 |
|---------|-----|----------|
| 账户名称 | 采购账户的名称 | 用户创建时输入 |
| 账户类型 | 账户对应的支付方式 | 系统预设或用户添加 |
| 账户余额 | 账户的当前资产余额 | 系统自动计算 |
| 收支类型 | 收入或支出类型 | 系统预设 |
| 收支金额 | 收入或支出的金额 | 系统记录 |
| 发生时间 | 收支发生的时间 | 系统记录 |
| 项目名称 | 收支对应的项目 | 系统预设或用户添加 |

## 3. 使用指南

### 3.1 基本操作流程

1. **进入模块**：从左侧菜单导航至 "ERP > 财务 > 采购记账" 模块
2. **查看账户**：系统会显示所有采购账户及其余额
3. **管理账户**：
   - 点击 "添加账户" 创建新账户
   - 点击账户卡片的更多按钮进行编辑、删除等操作
   - 点击 "查看删除列表" 恢复已删除的账户
4. **查看详情**：点击账户卡片查看该账户的收支趋势、分布和明细
5. **设置默认账户**：为每种支付方式设置默认账户

### 3.2 常见操作场景

#### 场景一：创建新的采购账户

1. 点击 "添加账户" 按钮
2. 在弹出的对话框中输入账户名称
3. 选择账户类型（支付方式）
4. 点击 "确认" 按钮完成创建

#### 场景二：设置默认账户

1. 在账户列表中找到需要设置为默认的账户
2. 点击账户卡片右上角的更多按钮
3. 选择 "设为该支付方式默认卡" 选项
4. 系统会自动将该账户设置为对应支付方式的默认账户

#### 场景三：查看账户收支明细

1. 在账户列表中点击需要查看的账户卡片
2. 右侧会显示该账户的收支趋势图、分布图和明细表
3. 可以在明细表中查看详细的收支记录

#### 场景四：恢复已删除的账户

1. 点击 "查看删除列表" 按钮
2. 在弹出的对话框中找到需要恢复的账户
3. 点击该账户对应的 "恢复" 按钮
4. 账户会被恢复到账户列表中

## 4. 常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 无法删除账户 | 账户存在余额 | 先将账户余额清零，再进行删除操作 |
| 无法取消默认账户 | 系统要求每种支付方式必须有一个默认账户 | 先为该支付方式设置另一个默认账户，再取消当前默认账户 |
| 添加账户失败 | 账户名称已存在 | 选择不同的账户名称 |
| 收支数据不显示 | 该账户暂无收支记录 | 进行相关的采购操作后，数据会自动更新 |
| 图表数据不准确 | 数据同步延迟 | 刷新页面或等待系统自动同步 |

## 5. 注意事项

- 采购记账模块会自动记录采购过程中的收支情况，无需手动录入
- 每个支付方式至少需要设置一个默认账户
- 删除账户前请确保账户余额为零
- 账户数据会实时更新，反映最新的财务状况
- 系统会自动生成日账单和月账单，方便财务分析
