# 发货费用分摊模块使用指南

## 1. 功能概述

发货费用分摊模块是Wimoor系统中用于管理和分析SKU头程费用的重要功能。该模块能够将发货总费用根据一定规则分摊到各个SKU上，帮助用户清晰了解每个SKU的头程成本，为产品定价和成本控制提供数据支持。

## 2. 页面访问

在系统左侧菜单栏中，依次点击：
**ERP管理** → **仓储管理** → **费用分摊**

## 3. 界面介绍

### 3.1 主界面布局

费用分摊模块主界面包含以下主要部分：

1. **标签页切换**：
   - **SKU头程**：展示SKU级别的费用汇总数据
   - **SKU头程明细**：展示具体发货单的费用分摊明细

2. **筛选条件区域**：
   - 本地仓库选择
   - 海外仓库选择
   - 日期范围选择
   - 承运商选择
   - 物流渠道选择
   - SKU/货件编码搜索框

3. **操作按钮区域**：
   - 导出：将当前筛选条件下的数据导出为Excel文件
   - 自定义列：自定义表格显示的列
   - 帮助文档：查看本帮助文档

4. **数据表格区域**：
   - 根据选择的标签页展示不同的数据内容

### 3.2 SKU头程标签页

该标签页展示SKU级别的费用汇总数据，主要字段包括：

| 字段名称 | 说明 |
|---------|------|
| 产品信息 | 包含产品图片、名称和SKU编码 |
| 产品负责人 | 负责该产品的人员姓名 |
| 发货数量 | 该SKU的总发货数量 |
| SKU头程费用(元) | 该SKU分摊到的总头程费用 |
| 平均单件头程费用(元) | 该SKU的平均单件头程费用 |
| 操作 | 点击"历史详情"查看该SKU的费用历史记录 |

### 3.3 SKU头程明细标签页

该标签页展示具体发货单的费用分摊明细，主要字段包括：

| 字段名称 | 说明 |
|---------|------|
| 产品信息 | 包含产品图片、名称和SKU编码 |
| 产品负责人 | 负责该产品的人员姓名和发货人 |
| 发货信息 | 发货仓库和发货状态 |
| 物流渠道 | 发货渠道和承运商 |
| 货件信息 | 货件编码和发货日期 |
| 收货站点 | 收货仓库名称 |
| 发货数量 | 该SKU在此次发货中的数量 |
| 实际结算重量 | 实际结算重量和SKU材积 |
| 货件运费(元) | 该货件的总运费 |
| SKU费用分摊 | SKU分摊到的费用和平均单件费用 |

## 4. 操作指南

### 4.1 数据筛选

1. **选择仓库**：
   - 点击"全部本地仓库"下拉框，选择特定的本地仓库
   - 点击"全部海外仓库"下拉框，选择特定的海外仓库

2. **选择日期范围**：
   - 点击日期选择器，选择起始日期和结束日期
   - 系统默认显示最近7天的数据

3. **选择承运商和物流渠道**：
   - 点击"选择承运商"下拉框，选择特定的承运商
   - 选择承运商后，系统会自动加载该承运商对应的物流渠道
   - 点击"选择物流渠道"下拉框，选择特定的物流渠道

4. **搜索SKU或货件编码**：
   - 在搜索框中输入SKU编码（SKU头程标签页）
   - 或输入SKU编码或货件编码（SKU头程明细标签页）
   - 点击搜索按钮或按回车键进行搜索

### 4.2 数据导出

1. 设置好筛选条件后，点击"导出"按钮
2. 系统会生成Excel文件并自动下载
3. 导出文件包含当前筛选条件下的所有数据

### 4.3 查看历史详情

1. 在"SKU头程"标签页中，找到需要查看历史详情的SKU
2. 点击该SKU所在行的"历史详情"按钮
3. 系统会弹出"SKU头程历史(每周费用分摊)"对话框
4. 对话框中包含：
   - 费用趋势图表：展示运费、发货数量和平均单价的趋势
   - 历史数据表格：展示该SKU的历史发货费用记录
5. 点击"关闭"按钮关闭对话框

## 5. 数据说明

### 5.1 费用分摊规则

系统根据以下规则进行费用分摊：

1. **按发货数量分摊**：根据每个SKU在发货单中的数量占比分摊费用
2. **按重量分摊**：根据每个SKU的重量占比分摊费用（部分场景）

### 5.2 关键数据字段解释

| 字段名称 | 计算方法 | 说明 |
|---------|---------|------|
| SKU头程费用 | 总运费 × (SKU发货数量 ÷ 总发货数量) | 该SKU分摊到的总头程费用 |
| 平均单件头程费用 | SKU头程费用 ÷ SKU发货数量 | 该SKU的平均单件头程费用 |
| SKU费用分摊 | 同SKU头程费用 | 明细中的SKU分摊费用 |
| SKU单件费用 | 同平均单件头程费用 | 明细中的SKU单件费用 |

## 6. 常见问题

### 6.1 为什么某些SKU没有费用数据？

可能的原因：
- 该SKU在所选时间段内没有发货记录
- 发货记录中没有填写运费信息
- 筛选条件设置不当，导致数据被过滤

### 6.2 如何理解费用趋势图表？

- **运费趋势**：展示每周的运费变化情况
- **发货数量趋势**：展示每周的发货数量变化情况
- **平均单价趋势**：展示每周的平均单件费用变化情况

### 6.3 导出的Excel文件包含哪些数据？

- **SKU头程**标签页导出：包含SKU、产品名称、负责人、发货数量、运费、本地SKU、单件平均费用等字段
- **SKU头程明细**标签页导出：包含本地SKU、编码、产品负责人、发货人、发货仓库、收货仓库、发货日期、发货承运商、发货渠道、产品名称、发货数量、实际结算重量、SKU预估重量、货件运费、SKU分摊费用、SKU单件费用等字段

## 7. 注意事项

1. **数据准确性**：请确保发货记录中的运费信息准确填写，否则会影响分摊结果的准确性
2. **筛选条件**：使用合适的筛选条件可以更快找到所需数据
3. **日期范围**：选择过大的日期范围可能会导致数据加载缓慢，建议合理设置日期范围
4. **数据导出**：导出大量数据时可能需要较长时间，请耐心等待
