# 差评管理模块帮助文档

## 1. 模块概述

差评管理模块是一个专门用于监控和管理亚马逊订单差评的功能模块，帮助卖家及时发现并处理负面评论，提升产品声誉和客户满意度。

### 1.1 主要功能

- 实时查看所有亚马逊订单的评论信息
- 按站点、订单、时间等维度筛选评论
- 查看评论详情，包括评分、评论内容和回复状态
- 快速识别低评分评论，及时进行处理

### 1.2 适用场景

- 卖家需要定期检查产品评论，及时发现并处理差评
- 客服团队需要统一管理和回复客户评论
- 产品运营人员需要分析评论趋势，改进产品质量

## 2. 操作指南

### 2.1 进入模块

1. 登录系统后，在左侧菜单中找到「客服」→「评论」→「差评管理」
2. 点击进入差评管理页面

### 2.2 查看评论列表

- 页面默认显示所有评论信息，按评论时间倒序排列
- 每条评论包含以下信息：
  - 站点信息（国家/地区）
  - 订单号及订单状态
  - 购买时间
  - 评论时间
  - 产品信息（图片、名称、SKU）
  - 评分（1-5星）
  - 评论内容
  - 回复内容
  - 客户邮箱

### 2.3 筛选评论

1. **按站点筛选**：通过顶部的站点选择器选择特定的亚马逊站点
2. **关键词搜索**：在搜索框中输入关键词，可搜索评论内容、产品名称或SKU

### 2.4 处理差评

1. 识别低评分（1-2星）评论
2. 查看评论内容，了解客户不满意的原因
3. 根据评论内容，制定相应的处理方案：
   - 联系客户了解具体问题
   - 提供解决方案或补偿
   - 在亚马逊后台回复评论

## 3. 常见问题

### 3.1 如何快速找到最新的差评？

- 页面默认按评论时间倒序排列，最新的评论会显示在最前面
- 可以通过评分筛选，只查看1-2星的评论

### 3.2 为什么有些评论没有回复内容？

- 回复内容需要在亚马逊后台进行回复，系统会同步亚马逊的回复状态
- 如果评论显示没有回复，说明尚未在亚马逊后台进行回复

### 3.3 如何导出评论数据？

- 目前系统暂不支持直接导出评论数据
- 可以通过截图或复制粘贴的方式保存重要评论信息

## 4. 注意事项

- 及时处理差评有助于提升产品排名和客户满意度
- 回复评论时要保持专业、礼貌的态度
- 定期查看评论可以帮助发现产品问题，及时改进
- 遵守亚马逊的评论政策，不要诱导客户修改或删除评论
