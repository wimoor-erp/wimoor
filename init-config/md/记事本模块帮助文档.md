# 记事本模块帮助文档

## 1. 模块概述

记事本模块是Wimoor系统中的一个内部工具，用于公司内部员工记录和管理各类笔记信息。该模块提供了富文本编辑、笔记管理、SKU绑定等功能，帮助用户高效管理工作笔记和相关信息。

**主要功能特点：**
- 富文本编辑器支持，可编辑格式化文本
- 笔记的增删改查操作
- 按标题、内容、SKU进行搜索
- 与产品SKU绑定功能
- 响应式布局，适配不同屏幕尺寸

**适用场景：**
- 记录工作备忘录
- 存储产品相关信息
- 团队内部知识共享
- 重要信息整理归档

## 2. 功能结构

### 2.1 前端功能结构

| 功能模块 | 子功能 | 描述 |
|---------|-------|------|
| 笔记列表 | 笔记展示 | 以列表形式展示所有笔记，支持分页 |
| 笔记列表 | 搜索功能 | 支持按标题、内容、SKU进行搜索 |
| 笔记列表 | 操作菜单 | 支持绑定SKU和删除操作 |
| 富文本编辑 | 文本编辑 | 支持格式化文本编辑，包含加粗、斜体等格式 |
| 富文本编辑 | 插入功能 | 支持插入图片、链接、表格等元素 |
| 笔记管理 | 新建笔记 | 创建新的空白笔记 |
| 笔记管理 | 保存笔记 | 保存当前编辑的笔记内容 |
| SKU绑定 | 选择产品 | 弹窗选择产品进行绑定 |
| SKU绑定 | 绑定操作 | 将笔记与选定的产品SKU关联 |

### 2.2 后端功能结构

| 功能模块 | 接口路径 | 方法 | 描述 |
|---------|---------|------|------|
| 笔记管理 | /api/v1/notepad/list | POST | 查询笔记列表，支持分页和搜索 |
| 笔记管理 | /api/v1/notepad/view | GET | 根据ID查看单个笔记详情 |
| 笔记管理 | /api/v1/notepad/save | POST | 保存笔记（新建或更新） |
| 笔记管理 | /api/v1/notepad/removeNote | POST | 删除指定笔记 |
| SKU绑定 | /api/v1/material/mark/notepad | POST | 将笔记与产品SKU绑定 |

## 3. 操作指南

### 3.1 前端操作流程

#### 3.1.1 查看笔记列表
1. 登录系统后，进入**系统管理** > **记事本**模块
2. 左侧面板会显示所有笔记的标题列表
3. 可以使用顶部搜索框按标题、内容或SKU搜索笔记
4. 点击笔记标题可在右侧编辑器中查看和编辑笔记内容

#### 3.1.2 创建新笔记
1. 在编辑器工具栏中点击**新建文档**按钮
2. 系统会清空编辑器内容，创建一个新的空白笔记
3. 在顶部输入框中输入笔记标题
4. 在编辑器中输入笔记内容，可使用各种格式化工具
5. 点击**保存**按钮保存新笔记

#### 3.1.3 编辑现有笔记
1. 在左侧列表中点击要编辑的笔记标题
2. 右侧编辑器会显示笔记内容
3. 对标题或内容进行修改
4. 点击**保存**按钮保存修改

#### 3.1.4 删除笔记
1. 在左侧列表中，找到要删除的笔记
2. 点击笔记右侧的**更多**按钮（三点图标）
3. 选择**删除**选项
4. 在弹出的确认对话框中点击**确定**

#### 3.1.5 绑定SKU
1. 在左侧列表中，找到要绑定SKU的笔记
2. 点击笔记右侧的**更多**按钮（三点图标）
3. 选择**绑定SKU**选项
4. 在弹出的产品选择对话框中，勾选要绑定的产品
5. 点击**确定**完成绑定

## 4. 常见问题与解决方案

### 4.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|----------|
| 笔记保存失败 | 标题已存在 | 更换笔记标题后重试 |
| 编辑器内容不显示 | 页面刷新问题 | 尝试刷新页面或重新打开编辑器 |
| 搜索无结果 | 搜索条件过于严格 | 尝试使用更宽松的搜索条件 |
| SKU绑定失败 | 未选择产品 | 确保在绑定对话框中选择了至少一个产品 |
| 笔记列表加载缓慢 | 笔记数量过多 | 尝试使用搜索条件过滤或分页查看 |

### 4.2 最佳实践

1. **笔记命名规范**：使用清晰、描述性的标题，便于后续查找
2. **内容组织**：合理使用编辑器的格式化功能，使笔记结构清晰
3. **定期整理**：定期清理过时笔记，保持笔记列表整洁
4. **SKU绑定**：对于产品相关笔记，建议绑定对应的SKU，方便后续关联查看
5. **搜索优化**：根据需要选择合适的搜索类型（标题、内容、SKU），提高搜索效率

## 5. 注意事项

1. **权限控制**：笔记默认按店铺ID隔离，确保用户只能访问自己店铺的笔记
2. **数据安全**：敏感信息请谨慎存储，系统会记录操作日志
3. **内容长度**：笔记内容建议控制在合理长度，避免影响加载速度
4. **浏览器兼容性**：建议使用Chrome、Firefox等现代浏览器，获得最佳编辑体验
5. **网络连接**：编辑过程中请确保网络连接稳定，避免数据丢失
