# 设置-账号管理模块帮助文档

## 1. 模块概述

账号管理模块是Wimoor系统中的核心功能模块之一，位于系统设置菜单下，主要用于用户管理个人账号信息、提升账号安全性、管理账号绑定关系等操作。该模块采用Vue 3 Composition API开发，结合Element Plus UI组件库，提供了直观、友好的用户界面。

## 2. 功能结构

账号管理模块包含以下主要功能：

| 功能模块 | 子功能 | 描述 |
|---------|--------|------|
| 个人信息管理 | 基本信息编辑 | 修改用户名称、联系电话、公司名称 |
| 个人信息管理 | 头像上传 | 上传和更新个人头像 |
| 账号安全 | 密码修改 | 修改登录密码 |
| 账号安全 | 账号更换 | 更换登录账号（手机号/邮箱） |
| 账号安全 | 邮箱管理 | 修改邮箱、激活邮箱 |
| 账号安全 | 账号迁移 | 将当前账号数据迁移至其他账号 |
| 微信绑定 | 微信绑定 | 通过扫码绑定微信账号 |
| 账号解绑 | 解绑管理 | 解绑已绑定的微信账号 |
| 账号注销 | 账号注销 | 注销当前账号 |

## 3. 操作指南

### 3.1 个人信息管理

#### 3.1.1 编辑基本信息

1. **操作步骤**：
   - 进入账号管理页面
   - 点击「编辑个人信息」按钮
   - 在弹出的对话框中填写新的用户名称、联系电话和公司名称（管理员账号）
   - 点击「提交」按钮保存修改

2. **注意事项**：
   - 用户名称不能为空
   - 管理员账号的公司名称不能为空
   - 修改成功后系统会自动刷新个人信息

#### 3.1.2 上传头像

1. **操作步骤**：
   - 点击个人头像区域
   - 选择本地图片文件（支持PNG、JPEG、JPG、BMP格式）
   - 系统自动上传并更新头像

2. **注意事项**：
   - 图片大小不能超过5000KB
   - 上传成功后系统会自动刷新头像显示

### 3.2 账号安全

#### 3.2.1 修改密码

1. **操作步骤**：
   - 在「账号安全」标签页中点击「修改」按钮
   - 输入原密码和新密码
   - 点击「确认」按钮完成修改

2. **注意事项**：
   - 新密码至少8位数以上，必须包含数字或字母组合
   - 两次输入的新密码必须一致

#### 3.2.2 更换账号

1. **操作步骤**：
   - 在「账号安全」标签页中点击「更换」按钮
   - 验证原账号密码和验证码
   - 输入新手机号/邮箱和验证码
   - 点击「下一步」完成更换

2. **注意事项**：
   - 更换账号需要验证原账号的密码和验证码
   - 新账号需要接收并验证验证码
   - 更换成功后系统会自动更新账号信息

#### 3.2.3 邮箱管理

1. **激活邮箱**：
   - 在「账号安全」标签页中点击「激活」按钮
   - 输入邮箱验证码
   - 点击「确认」完成激活

2. **修改邮箱**：
   - 在「账号安全」标签页中点击「更换」按钮
   - 输入新邮箱和验证码
   - 点击「确认」完成修改

#### 3.2.4 账号迁移

1. **操作步骤**：
   - 在「账号安全」标签页中点击「迁移」按钮（仅管理员账号可见）
   - 阅读并确认迁移风险
   - 输入目标账号和密码
   - 点击「确认」完成迁移

2. **注意事项**：
   - 账号迁移具有高风险，操作不可逆
   - 迁移后当前账号下的所有数据将转移至目标账号

### 3.3 微信绑定

1. **操作步骤**：
   - 进入「微信绑定」标签页
   - 使用微信扫码绑定

### 3.4 账号解绑

1. **操作步骤**：
   - 进入「账号解绑」标签页
   - 查看已绑定的账号列表
   - 点击对应账号的「解绑」按钮
   - 系统自动完成解绑操作

### 3.5 账号注销

1. **操作步骤**：
   - 进入「账号注销」标签页
   - 点击「注销」按钮
   - 确认注销操作
   - 系统自动完成注销并跳转到登录页面

2. **注意事项**：
   - 账号注销后相关数据将不再保留
   - 注销操作不可逆

## 4. 常见问题

### 4.1 个人信息修改失败

**问题**：点击提交后提示修改失败
**解决方案**：
- 检查用户名称是否为空
- 管理员账号检查公司名称是否为空
- 检查网络连接是否正常

### 4.2 头像上传失败

**问题**：上传头像后提示失败
**解决方案**：
- 检查图片格式是否支持（仅支持PNG、JPEG、JPG、BMP）
- 检查图片大小是否超过5000KB
- 检查网络连接是否正常

### 4.3 验证码获取失败

**问题**：点击获取验证码后没有收到
**解决方案**：
- 检查手机号/邮箱是否正确
- 检查是否在60秒内重复获取
- 检查网络连接是否正常

### 4.4 账号迁移失败

**问题**：账号迁移操作失败
**解决方案**：
- 检查目标账号和密码是否正确
- 检查目标账号是否存在
- 检查网络连接是否正常
