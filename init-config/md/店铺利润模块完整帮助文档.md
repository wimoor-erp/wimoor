# 店铺利润模块完整帮助文档

---

## 一、模块概述

### 1.1 功能定位

**店铺利润模块**是 Wimoor ERP 系统中用于统计和分析亚马逊店铺利润的核心模块。该模块按店铺/站点维度统计销售数据、费用支出和最终利润，支持多币种转换和数据导出功能。

### 1.2 核心特性

| 特性 | 说明 |
| :--- | :--- |
| **多店铺统计** | 支持按店铺分组统计利润 |
| **多币种支持** | 人民币、美元、站点币种 |
| **日期筛选** | 自定义日期范围查询 |
| **Excel导出** | 支持报表导出 |
| **实时汇率转换** | 自动进行货币转换 |

function getGroup(obj) {

  handleQuery()  // 触发查询
}

// 2. 日期变更处理
function changedate(dates) {
  profitGroupState.queryParams.fromDate = dates.start
  profitGroupState.queryParams.endDate = dates.end
  handleQuery()
}

// 3. 核心查询方法
function handleQuery() {
  profitGroupState.loading = true
  settlementAccRptApi.getSettlementOverAll(profitGroupState.queryParams).then((res) => {
    profitGroupState.loading = false
    // 格式化金额显示
    res.data.forEach(item => {
      item.principal = item.principal ? outputmoney(item.principal) : item.principal
      item.commission = item.commission ? outputmoney(item.commission) : item.commission
      // ... 其他字段格式化
    })
    profitGroupState.tableData = res.data
  })
}

// 4. 导出Excel
function downloadList() {
  settlementAccRptApi.downloadSettlementOverAll(profitGroupState.queryParams)
}

public Result<?> getSettlementOverAllAction(@RequestBody AmzSettlementDTO dto) {

    return Result.success(list);
}

```java
public List<Map<String, Object>> getSettlementGroupOverAll(Map<String, Object> map) {
    String shopid = map.get("shopid").toString();
    
    // 1. 获取结算公式配置
    AmzFinSettlementFormula formula = iAmzFinSettlementFormulaService.getAmzFinSettlementFormula(shopid);
    map.put("pricetype", formula.getPricetype());
    
    // 2. 设置结算日期参数
    this.setParamsSettlementDate(map);
    
    // 3. 查询基础结算数据
    List<Map<String, Object>> result = amzSettlementSummarySkuMapper.findSettlementOverall(map);
    
    // 4. 遍历处理每条记录
    for(Map<String, Object> item : result) {
        // 货币转换
        String fromcurrency = item.get("currency").toString();
        String tocurrency = systocurrency.isEmpty() ? fromcurrency : systocurrency;
        
        BigDecimal principal = exchangeRateHandlerService.changeCurrencyByLocal(
            shopid, fromcurrency, tocurrency, principal, 2);
        
        // 查询采购成本
        Map<String, Object> mymap = amzSettlementSummarySkuMapper.findSettlementSummary(map);
        BigDecimal price = new BigDecimal(mymap.get("price").toString());
        BigDecimal otherCost = new BigDecimal(mymap.get("otherCost").toString());
        price = price.add(otherCost);
        
        // 计算利润
        BigDecimal profit = setin.subtract(price);
        
        // 设置字段值
        item.put("principal", principal);
        item.put("price", price);
        item.put("profit", profit);
    }
    
    // 5. 按店铺分组汇总
    // ...
    
    return result2;
}
```

#### 3.3.2 利润计算公式

| 计算项 | 公式 | 说明 |
| :--- | :--- | :--- |
| **结算收入** | `setin` | 来自亚马逊结算报告 |
| **采购成本** | `price + otherCost` | 产品采购价 + 其他成本 |
| **利润** | `setin - price` | 结算收入 - 采购成本 |
| **其他费用** | `setin - storagefee - advfee - refund - fbafee - commission - principal` | 差额计算 |

### 4.1 完整数据流程

1. **前端请求**：用户选择店铺、日期、币种后点击查询
2. **参数校验**：后端校验参数，补全默认值（如默认近7天）
3. **数据查询**：从数据库查询结算数据
4. **货币转换**：根据目标币种进行汇率转换
5. **成本计算**：查询采购成本和其他费用
6. **利润计算**：结算收入 - 采购成本
7. **分组汇总**：按店铺分组并计算合计
8. **结果返回**：返回格式化后的利润数据

## 八、常见问题

### 8.1 数据为空怎么办？

**原因**：
- 未选择店铺/站点
- 日期范围内无结算数据
- 亚马逊授权过期

**解决**：
- 确认已选择店铺
- 调整日期范围
- 检查亚马逊授权状态

### 8.2 币种转换的汇率来源？

系统使用每日更新的实时汇率，汇率数据从第三方汇率服务获取。

### 8.3 如何查看单个SKU的利润？

店铺利润模块仅显示店铺级别数据。如需SKU级别分析，请使用"SKU利润"模块。
