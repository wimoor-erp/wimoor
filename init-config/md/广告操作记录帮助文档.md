# 广告操作记录帮助文档

## 1. 功能概述

广告操作记录模块是Wimoor系统中用于记录和查看广告相关操作历史的工具。该模块详细记录了所有广告活动、广告组、商品广告、关键词、定向等操作的历史记录，包括操作类型、操作人员、操作时间、操作内容等信息，为用户提供了完整的广告操作审计追踪功能。

### 1.1 模块价值

- **操作审计**：详细记录所有广告操作，便于审计和追溯
- **变更追踪**：追踪广告配置的变更历史，了解配置演变过程
- **责任明确**：记录操作人员信息，明确操作责任
- **问题排查**：当广告效果出现异常时，可通过操作记录排查原因
- **合规管理**：满足企业内部合规要求，提供操作证据

### 1.2 适用场景

- **日常管理**：定期查看广告操作记录，了解团队工作情况
- **问题排查**：广告效果异常时，查看近期操作记录
- **权限审计**：检查是否有未授权操作
- **合规检查**：满足内部或外部合规要求
- **操作培训**：通过查看历史操作，学习正确的操作流程

## 2. 页面访问

在系统左侧菜单栏中，依次点击：

**亚马逊** → **广告** → **广告操作记录**

### 2.1 权限说明

访问该模块需要具备以下权限：
- 亚马逊广告管理权限
- 对应店铺的广告操作权限

### 2.2 页面加载时间

- 首次加载时间：约3-5秒（取决于操作记录数量）
- 筛选条件变更后加载时间：约1-3秒

## 3. 界面介绍

### 3.1 主界面布局

广告操作记录模块主界面包含以下主要部分：

1. **筛选区域**：位于页面顶部，包含以下筛选条件：
   - **店铺/站点选择**：选择要查看的店铺和站点
   - **日期选择**：选择操作记录的时间范围
   - **搜索框**：搜索广告活动、广告组或商品名称

2. **操作记录列表**：位于页面中部，以表格形式展示操作记录，包含以下列：
   - **操作时间**：记录操作发生的时间
   - **广告活动(店铺/站点)**：显示广告活动名称、店铺名称和站点
   - **广告组**：显示广告组名称
   - **日志记录**：显示操作类型，点击可查看详细信息
   - **操作人员**：显示执行操作的人员
   - **备注**：显示操作备注，可点击编辑图标修改

3. **详情对话框**：点击日志记录链接时弹出，显示操作的详细信息

4. **备注编辑对话框**：点击备注编辑图标时弹出，用于修改操作备注

## 4. 操作指南

### 4.1 查看操作记录

#### 4.1.1 基础筛选

1. **选择店铺和站点**：在顶部的店铺/站点选择器中选择要查看的店铺和站点
2. **选择日期范围**：点击日期选择器，选择操作记录的时间范围
3. **执行筛选**：系统自动根据选择的条件筛选操作记录

#### 4.1.2 搜索操作记录

1. **输入搜索关键词**：在搜索框中输入广告活动名称、广告组名称或商品名称
2. **执行搜索**：系统自动根据关键词筛选操作记录

### 4.2 查看操作详情

1. **找到目标操作记录**：在操作记录列表中找到要查看详情的记录
2. **点击日志记录链接**：点击"日志记录"列中的蓝色链接
3. **查看详细信息**：在弹出的详情对话框中查看操作的详细信息，包括：
   - 操作类型
   - 操作时间
   - 操作人员
   - 操作对象
   - 变更前配置
   - 变更后配置
4. **关闭对话框**：查看完毕后，点击对话框右上角的关闭按钮

### 4.3 编辑操作备注

1. **找到目标操作记录**：在操作记录列表中找到要编辑备注的记录
2. **点击编辑图标**：点击"备注"列中的编辑图标
3. **输入备注内容**：在弹出的备注编辑对话框中输入备注内容
4. **保存备注**：点击"确定"按钮保存备注
5. **查看备注**：系统会更新操作记录列表中的备注内容

### 4.4 批量操作

该模块目前不支持批量操作，但可以通过筛选条件快速定位特定类型的操作记录。

## 6. 常见问题

### 6.1 操作记录不完整

**问题**：某些广告操作没有记录在操作记录中
**解决方案**：
- 检查操作是否成功执行（只有成功的操作才会记录）
- 检查操作权限（无权限的操作不会记录）
- 联系技术支持

### 6.2 操作记录显示异常

**问题**：操作记录显示的内容与实际操作不符
**解决方案**：
- 刷新页面，重新加载数据
- 检查筛选条件是否正确
- 联系技术支持

### 6.3 无法编辑备注

**问题**：点击备注编辑图标后无法编辑或保存备注
**解决方案**：
- 检查网络连接是否正常
- 检查是否具备编辑权限
- 刷新页面后重试
- 联系技术支持

### 6.4 操作记录加载缓慢

**问题**：操作记录列表加载缓慢
**解决方案**：
- 缩小日期范围，减少查询数据量
- 使用搜索功能，精确定位目标记录
- 检查网络连接是否正常
- 联系技术支持

## 7. 使用技巧

### 7.1 快速定位操作记录

1. **使用筛选条件**：结合店铺、站点和日期筛选，快速定位特定时间段的操作
2. **使用搜索功能**：输入广告活动或广告组名称，快速找到相关操作
3. **按时间排序**：系统默认按操作时间倒序排列，最新操作显示在最前面

### 7.2 有效利用操作记录

1. **定期审查**：定期查看操作记录，了解团队工作情况
2. **异常追踪**：当广告效果异常时，查看近期操作记录，排查是否与操作有关
3. **培训参考**：新员工可以通过查看操作记录，学习正确的操作流程
4. **合规检查**：定期检查操作记录，确保操作符合公司规定

### 7.3 备注管理

1. **添加详细备注**：对于重要操作，添加详细备注，说明操作原因和预期效果
2. **定期更新备注**：根据操作效果，及时更新备注，记录操作结果
3. **使用统一格式**：使用统一的备注格式，便于后续查询和分析
