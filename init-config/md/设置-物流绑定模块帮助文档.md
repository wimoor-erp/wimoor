# 设置-物流绑定模块帮助文档

## 1. 模块概述

设置-物流绑定模块是Wimoor系统中用于管理第三方物流商和海外仓API绑定信息的功能模块，通过该模块用户可以添加、编辑、启用、禁用第三方物流商和海外仓的API连接信息，实现与第三方物流系统的对接。

## 2. 功能说明

### 2.1 核心功能

- **API管理**：支持添加、编辑、启用、禁用第三方API（物流商/海外仓）
- **系统选择**：支持从预设的系统列表中选择API系统类型
- **授权信息配置**：支持配置API连接所需的AppKey、AppSecret和Token等授权信息
- **API列表**：展示已创建的API连接列表，包含系统名称、授权信息和状态

### 2.2 高级功能

- **系统过滤**：根据支持的功能类型（物流/海外仓）过滤API系统
- **批量管理**：支持批量启用/禁用API连接
- **API文档管理**：支持配置和查看API文档链接

## 3. 使用流程

### 3.1 添加API（物流商/海外仓）

1. **进入模块**：在系统菜单中找到"ERP" -> "第三方物流"模块
2. **点击添加**：点击页面左上角的"添加API（物流商/海外仓）"按钮
3. **选择系统**：在弹出的编辑对话框中，从"系统"下拉菜单中选择需要添加的API系统类型
4. **填写信息**：
   - 输入公司名称（必填）
   - 输入API连接地址（必填）
   - 根据系统要求，填写应用ID（AppKey）和应用密钥（AppSecret）
   - 根据系统要求，填写令牌（Token）
   - 输入API文档连接（可选）
5. **保存设置**：点击"保存"按钮保存API设置

### 3.2 编辑API信息

1. **进入模块**：在系统菜单中找到"ERP" -> "第三方物流"模块
2. **选择API**：在API列表中找到需要编辑的API连接
3. **点击编辑**：点击API连接右侧的"编辑"按钮
4. **修改信息**：在弹出的编辑对话框中修改API信息
5. **保存修改**：点击"保存"按钮保存修改

### 3.3 启用/禁用API

1. **进入模块**：在系统菜单中找到"ERP" -> "第三方物流"模块
2. **选择API**：在API列表中找到需要操作的API连接
3. **点击操作**：
   - 若API当前状态为启用，点击"禁用"按钮禁用API
   - 若API当前状态为禁用，点击"启用"按钮启用API
4. **操作确认**：系统会提示操作成功

### 3.4 查看API列表

1. **进入模块**：在系统菜单中找到"ERP" -> "第三方物流"模块
2. **查看列表**：页面会显示已创建的API连接列表，包含以下信息：
   - 名称：API连接的公司名称
   - 系统名称：API所属的系统类型，包含物流/海外仓标签
   - 授权信息：包含App Key、App Secret和Access Token等信息
   - Api：API连接地址
   - 操作：包含启用/禁用和编辑按钮

## 4. 常见问题

### 4.1 无法添加API怎么办？

- **问题原因**：可能是必填字段未填写完整，或API连接信息不正确
- **解决方法**：检查所有必填字段是否已填写，确保API连接信息的准确性

### 4.2 如何选择正确的系统类型？

- **系统类型**：系统下拉菜单中包含了预设的API系统类型
- **选择建议**：根据您要对接的第三方物流商或海外仓选择对应的系统类型
- **标签说明**：系统名称后会显示"物流"或"海外仓"标签，标识系统支持的功能类型

### 4.3 API连接失败怎么办？

- **问题原因**：可能是授权信息不正确，或API连接地址无法访问
- **解决方法**：检查AppKey、AppSecret和Token等授权信息是否正确，确保API连接地址可以正常访问

### 4.4 可以添加多少个API连接？

- **系统限制**：系统没有明确限制API连接数量，用户可以根据实际需求添加多个API连接
- **建议**：为了管理方便，建议只添加实际需要使用的API连接

## 5. 注意事项

### 5.1 操作注意事项

- **授权信息安全**：AppKey、AppSecret和Token等授权信息属于敏感信息，请妥善保管
- **API连接测试**：添加API连接后，建议进行连接测试，确保API可以正常访问
- **系统选择**：选择正确的系统类型，确保与第三方物流商或海外仓的系统匹配
- **信息更新**：当第三方物流商或海外仓的API信息发生变化时，及时更新系统中的API连接信息

### 5.2 技术注意事项

- **API版本**：确保使用的API版本与第三方物流商或海外仓的系统版本匹配
- **网络连接**：确保系统服务器可以正常访问第三方API的网络地址
- **权限设置**：确保第三方API账号具有足够的权限执行所需的操作
- **数据同步**：定期检查API连接状态，确保数据同步的及时性和准确性

### 5.3 安全注意事项

- **授权信息保护**：不要在公共场合或不安全的网络环境下配置API授权信息
- **定期更新**：定期更新API授权信息，特别是当员工离职或权限变更时
- **访问控制**：限制API连接信息的访问权限，只授予需要的人员
